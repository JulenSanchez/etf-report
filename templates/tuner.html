🛠 全流程总览 end-to-end ▶

每个调仓日，对当前勾选的 ETF 池执行以下流水线：

Phase 1: 因子计算

1. d₁ = 周线 EMA 偏离%（F1），r = 方向性量比（F3）

2. Z = N日累计对数收益的滚动 Z-score（F7）

Phase 2: 连续映射（每支 ETF 独立，不做截面排名）

3. d₁ → F1 = 11 + e−d₁/σ₁（sigmoid，σ₁ = F1 sigmoid 尺度）

4. r → F3 = 11 + e−ln(r)/σ₃（log-sigmoid，σ₃ = F3 指数尺度）

6. Z → F7 = 分段幂函数（|Z|≤k 幂次加速，|Z|>k 切线外延，k = 标准差倍数，t = 幂次）

Phase 3: 加权合成

7. 当前 Tuner 主路径：S = F1·w1 + F3·w3 + F7·w7

Phase 4: 仓位分配

9. 取 S 的 Top-N 支 ETF → Z-score 标准化 → Softmax 分配权重 wi（Ceff 由 C 动态灵敏度 根据 Top-N 分数离散度自适应调整：强共识放大 C，弱共识缩小 C）

10. 总仓位由 仓位控制 决定（沪深300 周线 MA 趋势信号，可叠加方向确认），与综合分独立

关键设计：连续映射保留因子绝对水平。牛市因子普遍高 → S 普遍高 → MA 信号满仓；熊市因子普遍低 → S 低 + MA 信号降仓。仓位由市场趋势驱动（MA），非自身 NAV 回撤驱动，避免频繁误触发踏空。

📈 F1 周线 EMA 偏离度 w1 ▶

Step 1: 计算 EMA

取周线收盘价序列，用指数移动平均公式计算：

EMAt = closet · α + EMAt−1 · (1−α)，其中 α = 2N+1，N = EMA 周期

Step 2: 算偏离度

d₁ = closet − EMAtEMAt × 100（偏离百分比，如 +5.2% 或 −3.8%）

Step 3: sigmoid 连续映射

F1 = 11 + e−d₁/σ₁，其中 σ₁ = F1 sigmoid 尺度

关键点标注见下图。σ₁ 越小越敏感（F1 sigmoid 尺度，当前值以左侧滑块为准）。

下图展示不同 σ₁ 下 d₁ → F1 的映射曲线：

与截面标准化的核心区别：sigmoid 保留绝对偏离水平。全市场-10%时F1≈0.13（低分），全市场+10%时≈0.87（高分），仓位控制可以识别牛熊。

📤 F3 自归一化量比 w3 ▶

Step 1: 取窗口数据

取最近 量比窗口 日（当前滑块值）的日线，计算每日涨跌 change = pct_change(close)

Step 2: 按涨跌分组

上涨日 = change > 0 的交易日，下跌日 = change < 0 的交易日

Step 3: 算方向性量比

优先用成交额（amount 列），无则退回成交量（volume 列）：

r = mean(vup)mean(vdown)，其中 vt = amounttavg(amountt−60:t−1)

边界：全涨无跌 → 封顶 3.0（强势上限），全跌无涨 → 0（极弱），无跌日且有涨 → 3.0

Step 4: 连续映射（log + sigmoid）

新版采用 log + sigmoid，让高位（r > 5）不再锁死：

ℓ = ln(r)（对 r < 1 取对称负 log：−ln(1/r)）

F3 = 11 + e−ℓ/σ₃，其中 σ₃ = F3 指数尺度

核心改进：旧版 r > 4 后基本饱和锁死，"温和放量"和"爆量"拉不开差距；新版让真正异动的标的能涌现出来。对照样本见下图标注点。

量比窗口：短窗口(5-10日)捕捉短期放量异动，长窗口(20-60日)看中期资金趋势。

F3 指数尺度 σ₃：越小越敏感（高位拉得更陡），越大越平滑。当前值以左侧滑块为准。

📉 F7 对数收益偏离 w7 ▶

Step 1: 日对数收益率

rt = ln(closet / closet-1)

对数收益可加、中轴天然为 0、涨跌对称。N 日对数收益之和 = ln(closet / closet-N)，N 由左侧“累计收益窗口”控制。

Step 2: N 日累计对数收益

cumt = Σi=0N-1 rt-i

窗口 = F7 对数收益偏离 权重滑块旁的左侧“累计收益窗口”滑块指定的 N 日；默认 20 日约 4 个交易周，滑块可扩到 60 日。

Step 3: 滚动 Z-score

对 cumt 序列，回溯 lookback = 250 日计算：

μt = mean(cumt-lookback … cumt-1)

σt = max(std(cumt-lookback … cumt-1), σfloor)

Zt = (cumt - μt) / σt

σfloor = 0.01，防止早期样本过少时 Z 值爆炸。数据不足 60 日 → Zt = NaN，该 ETF 跳过 F7（仅用 F1+F3 打分）。

Step 4: 分段映射（幂函数 + 切线线性外延）

核心变量：Z = Z-score，k = 切换阈值，t = 幂次

|Z| ≤ k 时：F7 = 0.5 − 0.5·sign(Z)·(|Z|/k)t（幂函数段，±k 内加速）

|Z| > k 时：F7 = f(k) + f′(k)·(Z−k)（切线线性外延，一阶连续不爆炸）

其中切线斜率 f′(k) = −t / (2k)。关键点标注见下图。

凸性设计：Z > 0（高估）→ 凸函数加速下跌；Z < 0（低估）→ 凹函数加速上涨。t 越大曲率越陡。

无上限惩罚：|Z| > k 后切线线性外延不截断，极端偏离可持续扣分/加分。

Step 5: 加权合成

综合分贡献 = F7 × F7 对数收益偏离权重

F7 是加法因子：最终综合分 = F1×w1 + F3×w3 + F7×w7。F7 是双向连续调节。

📐 仓位控制 conf ▶

将市场趋势信号直接映射为总仓位比例，控制"整体配多少仓"。

当前策略：MA趋势

基础信号：每期检查沪深300周线收盘价是否在自身MA趋势周期均线上方。

  • 沪深300 在周线MA 上方 → 总仓位 = MA上方仓位（以当前滑块为准）
  • 沪深300 在周线MA 下方 → 总仓位 = MA下方仓位（以当前滑块为准）
  • 若开启方向确认，则“是否在 MA 上方”和“MA 是否上行”必须一致才切换；不一致时维持上一状态。

设计理念：高常态仓位 + 极少触发的保护层

回测发现：熊市满仓（不降仓） vs 转现金的收益率差距并不大，说明只要选股本身有抗跌能力，不需要频繁调仓躲避。因此仓位控制的核心不是"精准逃顶"，而是"极少误触发 + 关键时刻提供保护"。

  • MA 上方使用左侧设定的进攻仓位，让选股策略充分运作
  • 仅当 MA 趋势翻空时才降到左侧设定的防守仓位，避免频繁操作导致踏空
  • 信号频率：约每年切换1-2次，极低频

为什么不用NAV驱动（旧方案已移除）

旧方案（regime / momentum_crash / quadratic）基于自身NAV回撤判断市场状态，问题在于：ETF组合的NAV波动与大盘并非同步，NAV回撤时可能大盘已企稳，NAV上涨时可能大盘已在崩盘。MA20是市场级信号，直接反映趋势，更可靠且不易误触发。

☷ F1 阶梯波 design ▶

F1 全程统一公式：F1(t) = (price(t) − EMA) / EMA。三种模式的区别只在 price(t) 冻结还是解冻。灰线（Base）永远冻结——price(t) = 上一完整周常量，信号是阶梯波。蓝线（Friday）非检查点冻结，启用的检查点交易日用当日 close 计算部分周 F1。紫线（Tue+Fri）周二检查点后，周三~四冻结（复用周二值），下一个检查点再更新。绿线（Daily）每个交易日都是检查点。多 bit 时中间日冻结是核心机制：不是"信号缺失"，而是主动拒绝低完整度估计。

⚖ 仓位分配 pos ▶

Step 1: Top-N 选股

取综合分最高的 最大持仓 支 ETF（以当前滑块为准）。

Step 2: Z-score 截面标准化

对全池子所有 ETF 的综合分算均值 μ 和标准差 σ，每支 ETF 转为 Z-score：

zi = (scorei - μ) / σ

快照表格中"离散度"列即 zi：>1.5 为显著领先，0~1.5 为中等，<0 为低于池子平均。效果：牛市整体分高、熊市整体分低时，自适应校准；极端离群值不霸榜。

Top-N 的 z-score 标准差 σtopN 即为动态 C 的"共识度"信号。

Step 3: Softmax 仓位分配

wi = ezi·CeffΣ ezj·Ceff

仓位集中度 C（base）控制分数差→仓位差的基准放大倍数。

Step 3.5: 动态 C（共识感知）

当 Top-N 的 z-score 离散度大（强共识）→ 自动放大 C；离散度小（弱共识）→ 自动缩小 C：

Ceff = Cbase × [1 + S × (σtopN − 0.5)]

其中 S = C 动态灵敏度。S=0 时禁用动态 C，等价于旧行为。σtopN = Top-N 的 z-score 标准差，反映"信号共识度"。

Step 4: 总仓位

MA趋势模式：沪深300在MA上方 → MA上方仓位，下方 → MA下方仓位。具体数值以当前 Preset/滑块为准。

posi = wi × total_pos

Step 5: 离散化

最大余数法 floor + 按余数补回，step = 离散化步长。若离散化后仍有残量，补给当前最大权重标的，保持总仓位不丢失。
