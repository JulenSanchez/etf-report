## 一、动机

现有优化指标（Sharpe、Sortino、总收益、MDD）都是**单条历史路径**上的统计量。凯利准则指出：真正应优化的是 **E[ln(终值)]**——对数财富的期望值，即长期几何增长率。

核心洞察：
- 单路径按总收益排名，选最优点 → 隐含假设"下次走法会和上次一样"
- Bootstrap 按中位数 ln(终值) 排名 → "在 1000 种不同的历史走法中，谁最稳"

## 二、方法

### 2.1 贝叶斯搜索（TPE）

不扫固定网格。Optuna TPE 采样器在 C∈[0.05,1.5], CS∈[0,5] 内做 50 次自适应搜索：
- 前 10 次（startup）：均匀随机探索
- 后 40 次：TPE 对已有结果拟合两个 KDE（l(x)=好组的分布, g(x)=坏组的分布），选下一点最大化 l(x)/g(x)

搜索自动收敛到有前途的区域。Top 15 全部聚集在 C∈[1.1,1.5], CS∈[3.6,4.8]。

### 2.2 Bootstrap 重采样

对每组 (C, CS)，从 6Y 回测（1453 个交易日）提取日收益率序列。有放回随机重排 1000 次，每次生成一条"替代历史"（日收益率排列不同，但集合相同），复利计算终值。

```
原始历史: r₁, r₂, r₃, ..., r₁₄₅₃            → 终值 = 16.3x
替代 #1:  r₅₃₇, r₁₂, r₁₂, r₈₉₁, r₅₃₇, ... → 终值 = 18.3x
替代 #2:  r₂₀₁, r₈₉₁, r₃, r₃, r₂₀₁, ...    → 终值 = 12.7x
...
替代 #1000                                     → 终值 = 9.8x
终值分布 → median, P5, P1, 毁灭概率
```

### 2.3 几何增长率

`g = ln(median_final) / years`，即年化连续复利增长率。`终值 = e^(g × years)`。

## 四、局限与诚实记录

### 4.1 本次实验范围内，凯利没有区分力

在 CS∈[0,5] 内，所有组合的毁灭概率 < 0.3%。Bootstrap 分布形状接近，中位数 ln(终值) 排名 ≈ 总收益排名。**凯利最优 = 单路径总收益最高。** 这不是凯利无用，而是策略在这个区域内太稳健——崩盘概率趋近零，bootstrap 只能确认单路径结论。

### 4.2 凯利真正的区分力在高 CS 区

CS=10（preset1）的几何增长率已经低于 CS=4.8（0.452 vs 0.491）。如果进一步推高 CS（20、50），按理论预期会出现：

- 单路径总收益：可能更高（历史上恰好抓对了）
- Bootstrap 中位数：显著更低（替代历史中崩盘次数多）
- **分歧在此出现**——凯利揭示单路径看不到的脆弱性

### 4.3 本次搜索上界（CS=5）碰巧在凯利最优点附近

如果上界设在 10 或 20，TPE 可能会收敛到更极端的 (C, CS) 组合——那些组合在单路径上极好看但 bootstrap 崩盘率高。这是后续实验方向。

## 五、结论

1. **(C=1.4, CS=4.8) 是 Kelly 最优组合**，在 bootstrap 1000 条替代历史中几何增长率最高
2. **Preset1 的 CS=10 属于"过度下注"**——f > f*，波动拖累超过收益增益
3. **C 应该更高（~1.4 而非 ~0.5）**——更高的基础集中度配合温和的 CS，同时保持稳健
4. **当前参数区域内策略极难死**——所有组合毁灭概率 < 0.3%
5. **完整验证凯利区分力需要探索 CS > 10**——那是凯利和单路径排名产生分歧的区域

## 六、后续方向

- **CS 扩展到 20-50**：找到"过度下注区"（f > 2f*）的起点
- **Bootstrap + Sortino 复合**：将凯利框架与用户提出的 Sortino+总收益均衡结合
- **滚动窗口凯利**：每 2Y 重估凯利最优 CS，验证是否漂移（WF 方向）
