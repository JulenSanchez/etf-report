# F2 主导策略参数优化

**日期**: 2026-05-19
**状态**: 规划中
**基准 preset**: `daily_aggressive`（F2=45%, F3=45%, F7=10%, C=0, score_band=3%）

## 固定项

| 参数 | 值 | 理由 |
|------|------|------|
| F2:F3:F7 权重比 | 45:45:10 | 锁定因子相对重要性，只调子参数 |
| MA trend 仓位控制 | bull=100%, bear=30%, period=26 | 已验证有效，不变 |
| max_holdings | 6 | 不变 |
| discretize_step | 0.05 | 不变 |
| rebalance_freq | daily | 不变 |

## 待优化参数

### Tier 1：因子灵敏度

| 参数 | 当前值 | 范围 | 常识判断 |
|------|--------|------|---------|
| F2 MA 周期 | 25 日 | 10~100 | 越短越灵敏→换仓多；越长越稳→接近周线 F1 |
| F2 sigmoid 尺度 | 8.0 | 3~15 | 越小越敏感，小偏离就拉开差距 |
| F3 量比窗口 | 20 日 | 5~40 | 越短噪声越大；越长信号越滞后 |
| F3 指数尺度 | 1.0 | 0.5~3.0 | 越大越激进，量比差异被放大 |
| F7 窗口 | 20 日 | 5~60 | 同上，且与 F3 窗口可能联动 |
| F7 t (曲率) | 15 | 3~25 | 越小越敏感，歧视极端值 |
| F7 k (敏感度) | 3.5 | 1~10 | 越大越激进 |

### Tier 2：仓位参数

| 参数 | 当前值 | 范围 | 常识判断 |
|------|--------|------|---------|
| C (集中度) | 0 | 0~3 | F2 比 F1 更灵敏→集中度可能需要更高 |
| score_band | 3% | 0~20% | 换仓频繁时提高 band 抑制噪音 |

## 探索计划

### Round 1: F2 MA 周期
- 固定其他，扫 F2 MA = [10, 15, 25, 40, 60]
- 期望：10-25 日区间有最优
- 评估：3Y Calmar + 1Y 验证

### Round 2: F2 sigmoid + F3 灵敏度
- 用 R1 最优，扫 (F2 sigmoid, F3 sens) = [(5, 0.8), (8, 1.0), (8, 1.5), (12, 1.0), (12, 1.5)]
- F2 更灵敏时 F3 也需要相应调整

### Round 3: 集中度 C
- 用前两轮最优，扫 C = [0, 0.5, 1.0, 1.5, 2.0]

### Round 4: score_band
- 用前三轮最优，扫 score_band = [0%, 3%, 5%, 10%]

### Round 5+: F7 参数微调
- 如果前三轮 F7 贡献显著，扫 F7 参数

## 停止条件
- 连续 2 轮改进 < 2pp 收益 且 MDD 不恶化
- 或最优值在可行域内部（非边界）

## 评估窗口
- 主窗口: 3Y (2023-05-19 ~ 2026-05-19)
- 验证窗口: 1Y (2025-05-19 ~ 2026-05-19) — 防过拟合
