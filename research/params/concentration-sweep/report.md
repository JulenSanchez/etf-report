# Concentration Sweep — daily_aggressive 浓度参数扫描

**日期**: 2026-05-20
**范围**: `daily_aggressive` preset, C ∈ {0, 0.3, 0.5, 0.7, 1.0}, 1Y + 3Y
**结论**: C=0.5 最优，已 promotion

## 动机

REQ-205 引入了 concentration 参数（softmax 陡峭度），`daily_aggressive` 初始设为 C=0（完全等权）。需要确认最优取值区间。

```
exp_scores = exp(z_scores * C)
weight_i = exp_scores_i / Σ(exp_scores)
```

- C=0 → 等权 Top-N
- C 越大 → 高分 ETF 权重越集中

## 方法

- **配置**: `daily_aggressive` preset（F1 周线 EMA 偏离 + F3 量比 + F7 对数收益偏离），MA Trend 信心函数
- **扫描**: C ∈ {0, 0.3, 0.5, 0.7, 1.0}
- **周期**: 1Y（2025-05-20~2026-05-20）、3Y（2023-05-22~2026-05-20）
- **指标**: 总收益、年化、MDD、Calmar、Sharpe、Sortino、换手率、佣金

## 分析

### 最优区间: C ∈ [0.3, 0.5]

两个周期交叉印证，峰值在 0.3-0.5。C=0.3 在 1Y 上略优（+0.59pp），C=0.5 在 3Y 上显著领先（+5.29pp）。长期证据权重更高。

### C ≥ 0.7 衰退

- C=0.7 在两个周期都垫底或接近垫底
- C=1.0 的 3Y 绝对收益回升至 +120.81%，但 Sharpe(1.15)/Sortino(1.62) 全组最差
- C=0.7 比 C=1.0 还差 — 中间地带存在非单调性："既不够分散也不够集中"

### 换手率单调递增

```
C:     0.0   0.3   0.5   0.7   1.0
1Y:    167 → 183 → 218 → 227 → 237
3Y:    470 → 508 → 584 → 624 → 658
```

每 +0.1 concentration ≈ 日均增加 6-7 次换仓。C=1.0 几乎每个调仓日都在交易。

### 为什么 C=0.5 比 C=0 好？

适度的集中允许策略在高信心 pick 上超配，而仍保持足够分散控制回撤。C=0 等权意味着第 1 名和第 6 名权重相同，浪费了信号区分度。C=0.5 在这两者之间找到了平衡。

### 为什么 C=1.0 反而退化？

Softmax 过陡 → 分数微小日间波动导致权重剧烈重排 → 追涨杀跌加速器 + 频繁换仓 + 集中踩雷同时发生。
