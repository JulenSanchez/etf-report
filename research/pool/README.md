## 字段参考

新增 ETF 到 `quant_universe.yaml` 时需填写的字段：

| 字段 | 必填 | 类型 | 说明 |
|------|------|------|------|
| `code` | ✅ | string | 6 位交易代码 |
| `name` | ✅ | string | 显示名称 |
| `market` | ✅ | string | `sh`=上交所 / `sz`=深交所 |
| `sector` | ✅ | string | 扇区归属，用于热力图配色和扇区过滤 |
| `bias` | 否 | bool | 扇区内偏好加成，给额外因子分数 |
| **`qdii`** | **见下** | bool | **QDII 跨境 ETF。标注后 Tuner 启用停牌检测，停牌时跳过全日量估算。只要是 QDII 跨境品种就必须标 true，不要遗漏。** |

> `qdii` 判断标准：该 ETF 是否投资境外市场且受 QDII 外汇额度限制？是 → `qdii: true`。
> 当前（2026-06）池中 3 支 QDII：513310（中韩半导体）、159509（纳指科技）、159941（纳指）。
> 新增纳指/日经/德国/沙特/标普/恒生类 ETF 时，一律检查是否需要标注 `qdii: true`。
