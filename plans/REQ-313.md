# REQ-313: real_margin 融资负债与利息引擎

**优先级**: 🟡 Normal | **目标版本**: v3.10.0 | **状态**: backlog | **创建**: 2026-06-24

## 目标

在回测引擎中新增 `real_margin` 账户模式：逐日计提融资利息、卖出时优先还款。

## 规格

- `account.mode = real_margin`
- `daily_rate = financing_rate_annual / 365`
- `interest = margin_debt * daily_rate` → `margin_debt += interest`
- 卖出时: `repay = min(sell_cash, margin_debt)` → `margin_debt -= repay`
- 利息进入 NAV（与 Stage A 的 pressure estimate 不同）

## 改动

`scripts/quant_backtest.py`：新增 `accrue_interest()` + `repay_debt()` 逻辑，在 Pass 2 逐日 NAV 循环中调用。
