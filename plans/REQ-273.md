## 动机

> 用户判定精算师"病入膏肓"。精算师在 1Y 窗口表现三派垫底，选股逻辑诡异，强趋势市中 F7 反转策略持续跑输。统计优势（6Y）≠ 实盘可跟随性——这个问题如果不诊断清楚，精算师这个"人设"在 v4 仪表盘里就没有说服力。

**现状问题**：
- 精算师 1Y：AR +118.8%/yr（三派垫底，禅修者 +122.4%，赌徒更高）
- 精算师 1Y 等权 S×C = 61.5（3Y=19.8, 6Y=11.7）——1Y 虽然绝对值高，但近期走弱
- "实盘可跟随性低"：用户明确表示实盘不会跟踪当前精算师
- F7 反转因子区分力极强（39.2，是其他因子的 5 倍），但可能在近期市场结构下失效
- 佣金吞噬严重（6Y 佣金 = 74% 本金）

**不做会怎样**：
- 精算师在 v4 仪表盘里是一个"数据漂亮但没人信的纸面人设"
- 可能在 v4.0.0 发布前就需要重新优化精算师参数
- 如果退化是结构性的，F7 因子本身需要重新审视

## 诊断范围

### 核心假说（待验证）

| # | 假说 | 预期信号 |
|---|------|---------|
| H1 | **ETF 池扩容稀释了 F7 信号** | 精算师在 25→45 支扩容后的选股质量下降，F7 在更多标的中"选错"的概率上升 |
| H2 | **市场风格转向强趋势** | F7 反转因子在趋势市中天然吃亏。如果 1Y 期间市场以趋势为主，精算师必然跑输 |
| H3 | **F7 因子时间序列衰减** | F7 的区分力（39.2）可能在全史上被早期数据拉高，近期区分力下降 |
| H4 | **佣金成本在短窗口更致命** | 1Y 窗口佣金占比可能更高（频繁调仓 + 短窗口 = 佣金来不及被收益覆盖） |
| H5 | **选股集中度失衡** | 精算师可能在少数"错误"标的上过度集中，单票风险放大 |

### 诊断步骤

1. **H1 池子稀释**：对比精算师在池子 25/34/40/45 支时的 1Y 滚动表现。如果扩容后指标单调下降 → H1 成立。
2. **H2 市场风格**：将 1Y 区间按市场风格分段。计算精算师在各风格下的相对表现（vs 禅修者/赌徒）。如果趋势段大幅跑输 → H2 成立。
3. **H3 F7 衰减**：F7 因子区分力的滚动 1Y 窗口。如果近期区分力下降 → H3 成立。
4. **H4 佣金拖累**：计算精算师 1Y vs 3Y vs 6Y 的佣金/总收益比。如果 1Y 比例最高 → H4 是恶化因素（但非根因）。
5. **H5 集中度**：精算师最近 1Y 的 P90 单票仓位、持仓集中度变化趋势。
