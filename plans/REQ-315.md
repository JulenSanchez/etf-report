# REQ-315: 追保/强平风险标记

**优先级**: 🟡 Normal | **目标版本**: v3.10.0 | **状态**: backlog | **创建**: 2026-06-24

## 目标

根据维持担保比例输出追保/强平风险标记。第一版只标记，不模拟券商强制卖出。

## 规格

```
if maintenance_ratio < 1.5: margin_call_flag = true   (追保预警)
if maintenance_ratio < 1.3: forced_liquidation_flag = true  (强平风险)
```

## 非目标

- 不模拟券商强平卖出行为
- 不连接真实券商账户

## 改动

`scripts/quant_backtest.py`：每日回测顺序第 8 步 `check_margin_status()` 输出标记。纳入 `AccountState` 序列。
