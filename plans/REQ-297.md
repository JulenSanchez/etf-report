# REQ-297: Stage A 杠杆风险指标与融资成本压力估算

**创建**: 2026-06-18 | **状态**: backlog | **优先级**: 🔴 P0 | **目标版本**: v3.9.0

## 背景

放开 `ma_bull_pos <= 2.0` 后，TR 会天然偏向更高杠杆。v3.9 Stage A 必须让杠杆风险显性化，避免“零成本杠杆”误导参数 promotion。

本需求负责合成杠杆的风险指标、暴露分布和融资成本压力估算。

## 目标

输出以下指标：

```text
avg_exposure
max_exposure
days_above_100
days_above_150
days_above_180
max_daily_loss
worst_month
leverage_usage_ratio
interest_drag_estimate
```

融资成本压力估算：

```text
avg_excess_exposure = average(max(exposure - 1.0, 0))
interest_drag_estimate = avg_excess_exposure * financing_rate_annual
```

注意：Stage A 的 `interest_drag_estimate` 只用于解释，不进入 NAV。

## 非目标

- 不做真实逐日融资利息。
- 不做维持担保比例。
- 不做强平模拟。

## 输出位置

- CLI 回测摘要。
- Tuner `/api/run` 返回结果。
- 后续 payload builder 可消费的 summary 字段。

## 预计改动区域

- `scripts/quant_backtest.py`
- `scripts/quant_tuner.py`
- `scripts/update_report.py` / payload helper
- `tests/test_quant_*`

## 验收标准

1. cash 模式指标存在但 exposure 恒定不超过 1.0。
2. synthetic_leverage 模式输出暴露天数分布。
3. `interest_drag_estimate` 可随 `financing_rate_annual` 改变。
4. 指标进入 Tuner 返回 payload。
5. `python -m pytest tests/test_quant_* -q` 通过。
