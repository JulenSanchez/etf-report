# REQ-389: 参数锁定单源治理

**优先级**: 🔴 P0 | **状态**: ✅ 已交付 | **目标版本**: v3.14.0 | **日期**: 2026-07-21

## 背景

参数锁定目前分散在四个地方：

| 位置 | 锁了什么 | 问题 |
|------|---------|------|
| `wf_phase1.py` GROUPS | N 不在任何组 | WF 脚本专用，其他模块不知道 |
| `DEFAULT_LOCK` | N=40, f7_lookback=250 | 只是默认值，不表达"不可改" |
| `tuner.html` `class="locked"` + `disabled` | slider 样式 | 手动写死，和逻辑无关 |
| `tuner.js` | `signal_steps: 40 /* locked */` | 硬编码值 |

改一次锁定状态要改 4 个文件。应该有一个单一真相源，所有模块从这里读。

## 设计

### `quant_contract.py` 新增 `LOCKED_PARAMS`

```python
LOCKED_PARAMS = {
    "signal_steps":   {"value": 40,  "reason": "纯摩擦参数。减小会隐式提升集中度，与 TB 越俎代庖。REQ-323 WF 诊断确认。"},
    "f7_lookback":    {"value": 250, "reason": "REQ-382 联合扫参确认 250 日最优，缩短无益。"},
    "conf_type":      {"value": "ma_trend", "reason": "仅 MA 趋势可用。旧信心函数已退役。"},
    "rebalance_freq": {"value": "daily",    "reason": "固定日调仓。周调仓因信号延迟已被淘汰。"},
    "benchmarks":     {"value": ["510300"], "reason": "固定沪深 300 单票。多指数投票未见稳健超额。"},
}
```

### 消费方

| 模块 | 改动 |
|------|------|
| `research_utils.py` `DEFAULT_LOCK` | N 和 f7_lookback 的值从 `LOCKED_PARAMS` 读（已有值保持） |
| `wf_phase1.py` | 加载时自动排除 `LOCKED_PARAMS` 中的 key |
| `PARAM_SCHEMA` | 锁定参数条目加 `"locked": true, "lock_reason": "..."` |
| `tuner.js` | `renderParamSchemaTable()` 读 schema 自动渲染 🔒 |
| `tuner.html` | 删除手动 `class="locked"` + `disabled` 属性 |
| `getParams()` / `setParams()` | 锁定参数从 `LOCKED_PARAMS` 取值，不读 DOM |

### 不改

- `PARAM_SCHEMA` 不删锁定参数的条目（参数存在，只是不可调）
- `_REQUIRED_PARAMS` 不变（锁定参数仍需校验存在性）

### 验证

| # | 操作 | 预期 |
|---|------|------|
| 1 | `pytest tests/ -x` | 全绿 |
| 2 | Tuner 加载 | 锁定参数显示 🔒 + tooltip |
| 3 | 切 preset | 锁定参数值不随 preset 变，保持锁定值 |
| 4 | 拖拽非锁定 slider → 回测 | 正常，锁定参数值正确传入 |
| 5 | `grep -r "signal_steps.*40"` | 只有 `LOCKED_PARAMS` 一处写 40 |

## 最终锁定清单

| 参数 | 值 | 锁模式 | 原因 |
|------|-----|--------|------|
| `signal_steps` | 40 | slider 灰 | 纯摩擦参数，减小隐式提升集中度 |
| `f7_lookback` | 250 | slider 灰 | REQ-382 扫参确认最优 |
| `conf_type` | ma_trend | enum 隐藏 btn-row | 仅 MA 趋势可用 |
| `rebalance_freq` | daily | enum 隐藏 btn-row | 固定日调仓 |
| `benchmarks` | [510300] | enum 隐藏 btn-row | 固定 HS300 单票 |
| `f1_active_days` | 1 | enum 隐藏 btn-row | 固定周五抢跑 |

## 两种锁模式

| 模式 | 触发条件 | 做法 | DOM 保留 |
|------|---------|------|---------|
| **slider 锁** | schema `unit` ≠ `"enum"` | input disabled + 灰 track + 🔒 | 是 |
| **enum 锁** | schema `unit` = `"enum"` | btn-row hidden + 🔒 | 是，解锁即恢复 |

## 新增锁只需

1. `LOCKED_PARAMS` 加一条
2. `PARAM_SCHEMA` 对应条目加 `locked: true, lock_reason: "..."`
3. 如果不在 schema，加一个条目 + `unit`
4. HTML DOM id 对齐为 `param-{key.replace('_','-')}`
5. 重启 Tuner
