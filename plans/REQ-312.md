# REQ-312: Runbook AI 可执行性优化

**优先级**: 🟡 Normal | **目标版本**: v3.9.1 | **状态**: backlog | **创建**: 2026-06-24

---

## 动机

审计发现 13 份 runbook 中 5 份 AI 无法完整执行、1 份完全不可执行。四个跨文档共性问题需要系统修复。

## Phase 1: 共性问题（所有文档）

### 1.1 统一触发词声明

每份 workflow 文档在第 1-3 行加 `> **触发词**:` 块（已有 4 份如此，其余 9 份补上）。

| 文件 | 触发词 |
|------|--------|
| overview.md | 无需（索引页） |
| optimization.md | 已有 `优化 <preset>` |
| pool-change.md | 已有 `换池` |
| preset-change.md | 已有 |
| screening.md | `筛选 ETF` |
| tuner.md | `启动 Tuner` `Tuner 白屏` `Tuner 报错` |
| data-fetch.md | `拉数据` `刷新 K 线` |
| daily-automation.md | `每日自动化` `收盘推送` `计划任务` |
| release.md | 已有 `发布` `提交` |
| stable.md | `更新 stable` `stable 同步` |
| audit.md | `审计` `项目审计` |
| v1-report.md | `更新报告` `生成正式页` |
| workflows.md | 无触发词——加 `<!-- AI: READ THIS FILE FIRST -->` |

### 1.2 统一"AI 暂停"标记

所有需要人类判断的步骤前加标准标记：

```markdown
> **AI 必须暂停**: <原因>。等待用户确认后继续。
```

影响文件：pool-change.md（无断崖下跌/扇区归属/短名审核）、preset-change.md（裁决流程）、screening.md（全部 5 条审阅规则）。

### 1.3 每条命令配预期输出

格式：

```markdown
python scripts/quant_backtest.py --preset gam-2 --start 2023-01-01
→ 预期: EXIT=0，NAV 曲线无 NaN，TR/MDD/Sharpe 打印在 stdout
```

影响文件：所有含 bash 命令的文档。

### 1.4 workflows.md 加 AI 指令

文件首行加 `<!-- AI: READ THIS FILE FIRST -->`。触发词列补同义词（如 `发布` 也匹配 `上线`、`deploy`）。加规则："AI: 匹配到触发词后必须完整阅读 owner 文档再行动，不可仅凭此索引执行。"

## Phase 2: 逐文件修复

### 2.1 screening.md — 从 Not Actionable → Partial

- 5 条审阅规则改为可量化阈值：
  - "重复暴露过高" → Jaccard > 0.5 标记
  - "足够历史数据" → history_days ≥ 250（引用 pool-change.md）
  - "足够成交额" → 日均成交额 > 2000 万
  - "QDII/商品/特殊策略" → 检查 `qdii` 字段和 sector
  - "短名清晰" → AI 建议，人确认（加暂停标记）
- 加 AI 暂停点：审阅阶段等待用户选择候选

### 2.2 pool-change.md — Partial → Executable

- "无断崖下跌" → 定量化：TR 下降 < 500bps，Sharpe 下降 < 0.2
- 扇区归属加 AI 暂停标记
- 短名审核加 AI 暂停标记（AI 建议，人确认）

### 2.3 preset-change.md — Partial → Executable

- 裁决流程加 AI 暂停标记
- Ledger 加示例行
- 落地失败回退流程

### 2.4 daily-automation.md — Partial → Executable

- 加交易日历检查步骤
- bat 文件常见失败模式 + 排查命令
- 正常状态描述（"16:05 后 index.html 应有今日日期"）

### 2.5 v1-report.md — Partial → Executable

- 区分"AI 执行什么"（一条命令）vs "内部做什么"（供调试）
- 加预期输出检查（文件大小、关键字符串）
