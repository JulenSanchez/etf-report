# REQ-346: 净值涨跌分布 UI

**优先级**: 🟡 P2 | **目标版本**: v3.11.0 | **状态**: 设计中

## 背景

Tuner 当前展示年化收益、最大回撤、Sharpe/Sortino 等汇总指标，但缺少对每日涨跌幅分布的直观展示。分布信息比单点统计量更丰富——同样的年化 50%，可能是每天稳稳涨 0.2%，也可能是几天暴涨+长期横盘。

## 目标

新增"涨跌分布"面板：展示回测周期内策略净值的每日涨跌幅统计分布。

## 核心设计

### 展示内容

- **直方图**：横轴 = 日涨跌幅区间（%），纵轴 = 该区间的交易日数
- **分段色阶**：大跌（红）、小跌（橙）、平盘（黄）、小涨（浅绿）、大涨（深绿）
- **汇总统计**：均值、中位数、标准差、偏度、峰度
- **极值标注**：最大单日涨幅、最大单日跌幅（日期 + 数值）

### 数据来源

`nav_df["nav"].pct_change()` — 已在回测引擎中计算（`daily_rets`），纯前端消费，无需后端改动。

### 区间分段建议

| 区间 | 标签 | 颜色 |
|------|------|------|
| < -5% | 暴跌 | 深红 |
| -5% ~ -2% | 大跌 | 红 |
| -2% ~ -1% | 小跌 | 橙 |
| -1% ~ +1% | 平盘 | 黄 |
| +1% ~ +2% | 小涨 | 浅绿 |
| +2% ~ +5% | 大涨 | 绿 |
| > +5% | 暴涨 | 深绿 |

### UI 位置

- 放在 NAV 图下方或指标卡片区（具体位置待布局框架 REQ-345 完成后确定）
- 使用 ECharts bar chart，色调与 Tuner 深色主题一致

## 改动范围

| 文件 | 改动 |
|------|------|
| `templates/tuner.html` | 新增涨跌分布 ECharts 面板 + 数据计算 |
| `scripts/quant_tuner.py` | 可能需要在 snapshot API 中暴露 `daily_rets` 序列 |

## 验收

1. 不同周期（1Y/3Y/6Y）切换时分布自动更新
2. 日K/周K 切换时分布使用对应的收益率序列
3. 极值标注显示具体日期
4. 色阶与 Tuner 深色主题一致
