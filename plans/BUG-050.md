# BUG-050: BUG-043 根因补完：softmax 数值稳定性

**严重度**: medium | **状态**: fixed | **发现日期**: 2026-07-09 | **关闭日期**: 2026-07-09

## 现象

`build_frontier_output` re-validation 阶段，gambler pool 中 18 个 `compute_frontier` 候选点只有 5 个幸存写入 `frontier_gambler.json`。被丢弃的 13 个点在重新跑回测时 `np.exp(z_scores * effective_c)` 产生 `overflow encountered in exp` 警告，softmax 输出 NaN，回测结果无效，MDD 超出 `[-50, -15]` 范围被过滤。

Optimizer 日志打印 `Frontier: 18 points`（compute_frontier 初选数），但文件实际只有 5 点——AI 据此误报"前沿 6→18 点 +191%"，用户在 Tuner 浏览器只看到 5 个点才发现问题。

## 根因

`quant_backtest.py:910` 的 softmax 防溢出实现：

```python
_raw = z_scores * effective_c
_raw = np.clip(_raw, -700.0, 700.0)  # exp(709) ~ 8e307, 接近 float64 max
exp_scores = np.exp(_raw)
```

`effective_c = concentration × max(1.0 + c_sensitivity × (dispersion - 0.5), 0.1)`，高 c_sensitivity 参数（如 c_sensitivity=28、concentration=26.1，与 BUG-043 同一组极端参数）下 effective_c 可达 5-8，`z_scores × effective_c` 轻易超过 700。clip 到 700 时 `exp(700) ≈ 1e304`，多个 ETF 同时接近上限时 `exp_scores.sum()` 数值不稳定，仍可能溢出或产生 NaN。

### 计算链

```
高 c_sensitivity (≥28) + 高 concentration (≥26)
  → effective_c = 5~8 倍正常值
  → z_scores × effective_c → 部分值 > 700
  → np.clip(-700, 700) 仍接近 float64 上限
  → softmax 输出含 NaN 或极端峰值
  → 回测 NAV 含 NaN
  → build_frontier_output re-validation 丢弃该点
```

### 与 BUG-043 的关系

BUG-043 同根因（高 c_sensitivity 极端参数），但症状层不同：

- **BUG-043**：softmax 权重集中 → `target_positions` 全零 → `idxmax()` 返回 NaN → `KeyError: nan`。修了 `idxmax()` 防护（症状层）。
- **BUG-050**：softmax 本身溢出 → NaN。修了 softmax 实现（根因层）。

本 bug 是 BUG-043 根因的补完修复。

## 修复

`quant_backtest.py:910-913` 改为 stable softmax（减最大值再 exp，数学等价但不溢出）：

```python
# Stable softmax: subtract max before exp to prevent overflow.
# exp(710) > float64 max → inf, but exp(x - max(x)) is always ≤ 1.
_raw = z_scores * effective_c
_raw = _raw - _raw.max()  # ← 减最大值，永不溢出
exp_scores = np.exp(_raw)
softmax_w = exp_scores / exp_scores.sum()
```

`exp(x - max(x))` 永远 ≤ 1，`sum` 永远 ≥ 1（至少一个 `exp(0) = 1`），不依赖 clip 阈值，对任意 c_sensitivity 都稳定。

### 语义说明

减最大值是 softmax 的标准稳定化技巧，数学上与原式完全等价（分子分母同除 `exp(max)`，约掉）。不改变 c_sensitivity 的实际作用强度，只改数值稳定性。

## 遗留
