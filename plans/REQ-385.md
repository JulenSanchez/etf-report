**优先级**: 🔴 P0 | **状态**: ✅ 已关闭（决策落定，并入 REQ-323） | **日期**: 2026-07-20

## 背景

Walk-Forward Phase 1 跑 3 步，每步产出样本外 AR。需要合并指标评判 WF 轨迹好坏。

## 决策：C — 全程拼接回测（2026-07-20）

```
1. 用 P1 跑 2023-06~2024-06 → NAV 序列 1
2. 用 P2 跑 2024-06~2025-06 → NAV 序列 2
3. 用 P3 跑 2025-06~2026-06 → NAV 序列 3
4. 拼接成连续 NAV → 读总 AR / MDD / Sharpe
```

理由：最接近实盘体验，MDD 是连续曲线真实回撤而非 3 段合并，拼接点天然对齐调仓日。

Phase 2 前沿 X 轴同样用拼接 NAV 的连续 MDD。

拼接点过渡策略（直接换仓 vs 渐进）作为 REQ-323 实现细节，不在方法论层面决策。
