## 动机

当前仓库将作为 Agent 技能被安装到 CodeBuddy、Claude Code 或其他 Agent 环境。

因此对外提交不仅要考虑“人能不能跑”，还要考虑“AI 第一次读仓库时能不能快速理解入口、配置与发布链路”。

## 实施步骤

### Step 1: 整理发布白名单
- 明确这次 force push 应保留的目录和文件
- 明确不应提交的开发治理与私有配置

### Step 2: 设计 AI 首读路径
- 明确 Agent 安装后优先阅读哪些文档
- 明确改配置、接发布链路、执行主流程的入口

### Step 3: 文档导航优化
- 在对外文档中补充 Agent 技能场景的说明
- 让 AI 更容易从 README / SKILL 中抓到操作顺序
