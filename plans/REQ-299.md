# REQ-299: 三派合成杠杆参数优化与 promotion 裁决

**创建**: 2026-06-18 | **状态**: backlog | **优先级**: 🟠 High | **目标版本**: v3.9.0

## 背景

Stage A 打开 `ma_bull_pos <= 2.0` 后，必须重新评估三派风险预算。杠杆不是全局参数，而是人格策略的风险表达。

## 目标

分别优化：

```text
act-1 leverage profile
zen-1 leverage profile
gam-1 leverage profile
```

建议搜索上限：

```text
act-1: ma_bull_pos 1.0 ~ 1.5
zen-1: ma_bull_pos 1.0 ~ 1.3
gam-1: ma_bull_pos 1.0 ~ 2.0
```

候选命名：

```text
act-margin-1
zen-margin-1
gam-margin-1
```

## 优化约束

硬过滤建议：

```text
MDD <= 35%
max_daily_loss <= 8%
worst_month >= -18%
```

排序目标建议：

```text
score = Sortino * Calmar
```

或：

```text
score = annual_return / abs(max_drawdown)
```

## 裁决口径

只允许三类结论：

```text
reject
research-only
promote-with-guardrails
```

## 非目标

- 不自动覆盖 act-1 / zen-1 / gam-1。
- 不只按 TR 排名。
- 不使用无风险提示的零成本杠杆结论。

## 预计改动区域

- `scripts/quant_tuner.py` 或优化脚本
- `research/params/`
- `config/quant_universe.yaml`（仅 promotion 后）
- `docs/runbook/v2-quant/optimization.md`

## 验收标准

1. 三派各有完整合成杠杆优化报告。
2. 报告包含 exposure 分布和融资成本压力估算。
3. 报告包含硬过滤失败项，不只列最优项。
4. 明确给出 reject / research-only / promote-with-guardrails。
5. 任何 promotion 都必须保留原现金账户 preset 可回退。
