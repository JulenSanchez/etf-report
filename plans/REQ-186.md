# REQ-186: ETF 交易佣金模拟

**优先级**: 🟠 High
**状态**: done
**创建日期**: 2026-04-30
**最后活动**: 2026-04-30

## 动机

当前回测零交易成本，日调仓换仓频繁时高估了实际收益。ETF 无印花税，但佣金+过户费仍存在，应在回测中扣除以反映真实收益。

## 方案

ETF 单次交易费率 0.026%（买入+卖出各一次 = 双向 0.052%），在每次成交时从现金中扣除。

- 买入：`cash -= buy_value * commission_rate`
- 卖出：`cash += sell_value * (1 - commission_rate)`

佣金率默认 0.00026（万0.26），存入 `quant_universe.yaml`。

## 文件影响

| 文件 | 改动 |
|------|------|
| `quant_backtest.py` | 买入/卖出时扣除佣金；输出总佣金 |
| `quant_tuner.py` | 同步佣金逻辑；metrics 新增 totalCommission |
| `tuner.html` | 展示总佣金信息 |
| `quant_universe.yaml` | 新增 commission_rate 配置 |

## 验证

- ✅ 佣金=0 时结果与当前完全一致（逻辑上 commission_rate 默认为 0）
- ✅ 佣金>0 时年化收益下降，换仓越多下降越大（日调仓佣金 0.74-2.20% vs 周调仓 1.18%）
- ✅ Tuner API 返回 commissionPct + totalCommission
- ✅ UI 展示交易佣金 metric card

## 结论（2026-04-30）

佣金模拟已端到端落地。关键数据：
- 日调仓+sb=5%：佣金 0.74%（36次换仓）
- 日调仓+sb=0：佣金 2.20%（129次换仓）
- 周调仓：佣金 1.18%（43次换仓）

分数带有效控制了佣金损耗，日调仓+sb=5% 的佣金甚至低于周调仓。
