## 动机

> 在 ETF 卡片加基本面估值水位：**当前值 + 百分位数值 + 五档判词**，让用户快速感知"这支现在贵不贵"。

**现状问题**：
- 报告页只有行情/持仓/技术走势，缺基本面
- 无法回答"沪深300 PE 15 倍到底是高还是低"这类最基础问题
- 月度再平衡（ashare-trading-system 技能）缺估值输入

## 方法论沉淀

**完整方法论见 `docs/VALUATION_METHODOLOGY.md`**（11 章投教级文档）。

本需求的关键决策：
- **单一主指标**：每支 ETF 锁定一个指标（PB 或 PE），不做多指标展示
- **方法 B'（混合演化）**：锚点规则即刻可用 + 每日增量积累真实历史 + 样本足够时自动切换到严格百分位
- **五档判词**：20/40/60/80 分位（极低/低/合理/偏高/极高）
- **完全透明**：UI 角标标注置信度（规则/混合/历史）

## 实施步骤

### Step 1: 锚点表沉淀（2h）

- AI 基于 2015-2025 行业 PE/PB 走势 + 标志性事件（2020 疫情/2021 锂电顶/2024 医药底等）研究
- 为 6 支 ETF 各沉淀 5 点锚点（extreme_low / low / median / high / extreme_high）
- 每个锚点附 `reference_events` 说明依据
- 产出：`config/valuation_anchors.yaml`
- 验证：锚点值与集思录/理杏仁公开页面历史图感交叉校准

### Step 2: 估值引擎（2h）

- 新建 `scripts/valuation_engine.py`
- 核心函数：
  - `load_anchors(yaml_path)` - 读锚点
  - `rule_based_percentile(current_value, anchors)` - 分段线性插值
  - `time_series_percentile(current_value, history)` - 标准时间序列百分位
  - `blended_percentile(current_value, history, anchors)` - 根据样本数自动切换（<100 纯规则 / 100-500 混合 / >500 纯历史）
  - `verdict(percentile)` - 五档判词
- 单元测试：`tests/test_valuation_engine.py`

### Step 3: 数据获取 + 增量累积（1.5h）

- 新建 `scripts/valuation_fetcher.py`
- 用 AkShare `stock_zh_index_value_csindex` 拉当前 PE/PB（20 天快照）
- **关键**：每日追加到 `data/valuation_history/<index_code>.csv`，去重保留唯一日期
- 指数映射表 `config/etf_index_mapping.yaml`：
  - 512400 → 000819 申万有色
  - 513120 → 港股创新药（需查真实指数）
  - 512070 → 真实跟踪指数（非 399973 国防，PoC 发现映射错）
  - 515880 → 931160 中证全指通信设备
  - 159566 → 真实跟踪指数（非 980032，PoC 404）
  - 159865 → 930707 中证畜牧养殖
- 异常处理：接口失败 / 指数未覆盖 / 港股无 csindex 源 → 降级到"仅锚点"模式

### Step 4: 集成到 update_report（1h）

- `update_report.py` 日更流程新增 valuation 步骤
- 输出 schema:
```json
{
  "etf_code": "512400",
  "index_code": "000819",
  "primary_metric": "pb",
  "current_value": 2.35,
  "percentile": 42,
  "verdict": { "label": "合理", "emoji": "⚪", "color": "gray" },
  "confidence": "rule",
  "sample_days": 15,
  "window_note": "锚点规则 · 仅 15 天真实样本"
}
```
- CLI 开关：`--skip-valuation`

### Step 5: 前端水位条 + 透明度角标（1.5h）

- `index.html` ETF 卡片新增"估值水位" section
- 布局：
  - 当前值大字（如"PB 2.35"）
  - 水位条（0-100% 彩色渐变）+ 当前位置指针
  - 判词色块
  - 置信度角标（右上角："规则估算"/"混合 N 天"/"历史 N 天"）
  - Hover tooltip：完整 window_note
- 色值复用现有 CSS variables
- 链接到 `docs/VALUATION_METHODOLOGY.md`

### Step 6: 验证 + 文档微调（1h）

- 端到端跑 `update_report.py`
- 抽 2 支 ETF 与集思录公开页面交叉校验（锚点版允许 ±10%）
- README.md 特性清单补一行
- `docs/VALUATION_METHODOLOGY.md` 补充第 9 节实际锚点值
