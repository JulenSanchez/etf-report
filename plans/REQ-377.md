# REQ-377: 因子驱动力日度诊断 — 截面离散度 + 主导因子识别

**优先级**: 🟡 P1 | **状态**: ✅ 已交付 | **目标版本**: v3.14.0

## 背景

当前 Tuner 的调仓快照展示单日各 ETF 的 F1/F3/F7 分值，但没有汇总"这一天整体是什么行情"的指标。用户观察到某回测日 8 支 ETF 的 F7 超过 40（其中 5 支 >100，最高 587），感到这是极端行情，但无法判断历史上属于什么水平——是策略遇到了前所未见的情况、即将失效，还是正常的尾部事件。

核心需求：**给每一天一个"因子驱动力画像"**，回答三个问题：
1. 今天是谁在主导？F1（趋势）、F3（量比）、还是 F7（极端偏离）？
2. 今天的极端程度在历史上排第几？
3. 极端日是否过于频繁（→ 策略太敏感，需要调参）？

## 设计

### 日度因子截面统计

对每个交易日 t，在全池 ETF 上计算每个因子的截面分布：

| 指标 | 公式 | 含义 |
|------|------|------|
| μ_F1(t), μ_F3(t), μ_F7(t) | 全池均值 | 因子整体水位 |
| σ_F1(t), σ_F3(t), σ_F7(t) | 全池标准差 | 因子分化程度 |
| max_F7(t) | 全池最大 F7 | 最极端个股 |
| pct_F7_gt_40(t) | F7 > 40 的 ETF 占比 | 极端 F7 覆盖面 |
| pct_F7_gt_100(t) | F7 > 100 的 ETF 占比 | 超极端 F7 覆盖面 |

### 主导因子识别

定义"因子支配度" = 该因子截面标准差的相对强度：

```
F1_dominance(t) = σ_F1(t) / (σ_F1(t) + σ_F3(t) + σ_F7(t))
F7_dominance(t) = σ_F7(t) / (σ_F1(t) + σ_F3(t) + σ_F7(t))
```

直观含义：
- F7_dominance 高 → F7 在 ETF 之间拉开巨大差异 → 市场极度分化 → 往往意味着反转
- F1_dominance 高 + F7_dominance 低 → 趋势平稳延续 → "躺着赚钱的好日子"

### 历史百分位

对每个日度指标，计算其在回测窗口内的滚动百分位：

```
F7_max_percentile(t) = percentile(m_F7_max(t-250:t), m_F7_max(t))
```

用户看到"F7 最高 587"，查表可知"这在过去 250 个交易日中排前 3%"，就不再恐慌。

### 极端日频率

定义"极端日" = F7_max_percentile > 90%（或其他阈值），统计滚动窗口内极端日占比：

```
extreme_day_ratio = count(extreme days in window) / window_length
```

如果 250 天里有 60 天是极端日（24%），说明策略太敏感，F7 参数（t/k/window）可能需要收敛。

## Tuner 集成

### 左侧面板：因子驱动力卡片（新增 section）

在回测结果区新增一个小节，展示当前回测窗口的驱动力摘要：

- **主导因子饼图/条形**：F1/F3/F7 主导的天数占比

### 右侧面板：驱动力时序图

## 前期调研（2026-07-16）

用 gam-0 跑 6Y 回测（2020-2026），`return_details=True` 提取每调仓日的全池 ETF 映射因子分（f1/f3/f7，0-100 量程），计算截面离散度 + 主导因子。脚本 `temp_scripts/temp_script_20260716_factor_dispersion.py`，输出 CSV `research/params/factor_dispersion.csv`（856 行）。

### 数据源

`run_backtest(return_details=True)` → `signal_history[].detail` 包含每 ETF 的 `f1`/`f3`/`f7`（映射后的 0-100 分值）。直接取用，不需要改引擎。

### F7 极端度分布

```
P50=51   P75=152   P90=413   P95=582   P99=1098   MAX=1915
```

> 注：F7 映射函数的切线线性外延（|Z| > k）会产生 >100 的值，这是设计行为而非 bug。

用户观察到的"F7 最高 587"约在 P95——比 95% 的日子极端，但历史上还有 ~40 天更极端。不是前所未见，而是 5% 尾部事件。

### 因子主导权

```
F1 主导: 548 天 (64%)  — avg F7_max = 53   （趋势平稳，F7 安静）
F3 主导:   0 天 (0%)   — 量比从未成为主导力
F7 主导: 308 天 (36%)  — avg F7_max = 336  （极端分化，F7 接管）
```

F1 是默认主导力（64%），但当 F7 接管时极端度跳升 6 倍（53 → 336）。验证了用户直觉：F7 飙高 = 行情走到极端。

### 极端日年分布

```
2023:  2 天
2024: 40 天  ← 924 行情 + 8 月暴跌 + 10 月巨震
2025: 12 天
2026: 32 天
```

与已知行情结构吻合。2024 年是中国股市波动率最高的一年，极端日数量也应声跳升。

### 敏感度信号

F7_max > 100 且 F7 主导（dom > 40%）的天数 = **261 天（30%）**。这个比例偏高，暗示 gam-0 的 F7 参数（t=7, k=3.0）可能偏敏感——三分之一的调仓日 F7 在强力驱动。可作为调参反馈指标：扫描参数时对比极端日频率。

### 技术结论

1. **主导因子识别有效**：截面标准差比能区分"F1 趋势日"和"F7 极端日"，两组的 avg F7_max 差距 6 倍
2. **历史百分位可行**：`run_backtest(return_details=True)` 直接提供所需数据，约 30 行 Python 完成全量计算
3. **极端日过多是调参信号**：30% 的 F7 主导+极端天数 > 合理范围，可作为敏感度诊断指标
4. **F3 从未主导**：量比信号的截面分化程度远低于 F1 和 F7，不需要为 F3 做独立诊断

### 二次研究：反转时间偏差（2026-07-19）

定义反转点 = NAV 局部峰/谷（前后 20 天 + 后续 5% 回撤/反弹）。分析 σ_F7 峰值相对反转点的时间偏差：

```
σ_F7 峰值 vs 反转点: 均值 +0.2天 中位 0天  标准差 17天
领先(负): 21次  同步: 3次  滞后(正): 19次
```

- **偏差均值 ≈ 0**：σ_F7 既不过敏也不迟钝，均匀分布在反转点两侧 ±2-3 周内。
- **顶/底差异小**：峰(顶) 均值 -1.0d，谷(底) 均值 +1.7d。
- **σ_F7 不能预测方向**：与前瞻收益 r≈0，与前瞻回撤 r≈0。它是同步指标，记录已经发生的市场撕裂。

### 三次研究：F7 量程不对称问题（2026-07-19）

2026-07 实盘诊断发现：创新药 F1 持续走强 (33→73) 但 F7 平静 (50)，电网设备 F1 崩跌 (61→16) 但 F7 爆炸 (+50→+752)。策略选了电网设备，因为 F7=661 × w7=16% = 106 分，单项超过创新药全部。

**根因**：F7 切线外延无上限（可达 750+），F1 硬顶 0-100。量程不对称导致任何 w7 > 0 都会在极端行情下让 F7 主导一切。

### 四次研究：Z-score 爆炸的数学根因（2026-07-20）

以电网设备(159326) 2026 年崩跌为例，逐日检查 Z-score 的分子(20d cum)、分母(μ_250d, σ_250d)：

```
上涨期: μ=+0.022→+0.054  σ≈0.06（窄幅稳步上涨 200 天）
7/03:   20d cum=-0.07  μ=+0.055  σ=0.067  Z=-1.8  F7≈50（刚破位）
7/10:   20d cum=-0.16  μ=+0.052  σ=0.071  Z=-3.1  F7=108
7/17:   20d cum 继续恶化  μ≈+0.050  σ≈0.07  Z=-8~-10  F7=750+
```

μ 从 0.055 降到 0.050 用了 14 天——250 日历史惯性太强。σ 被牛市压制在 0.06-0.07。分子变大(跌了)、分母不变(μ 高 σ 窄)→ Z-score 爆炸。

**解决方案**：暴露 `F7_lookback`（归一化窗口）作为可调参数，与 `F7_window` 联合寻优。→ REQ-382
