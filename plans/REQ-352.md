# REQ-352: 反事实对比 — band 效果的直观展示

**优先级**: 🟢 P3 | **目标版本**: v4.0.0+ | **状态**: 设计中

## 背景

C/CS/B/BS 参数体系已落地，但参数效果不直观。band=0.03 和 band=0 的持仓差异有多大？用户只能在脑子里抽象比较，看不到具体差距。

## 目标

在调仓快照中展示"如果关闭 band（band=0），当前持仓会是什么"的反事实对比。

## 方案

- 前端在渲染信号时，用 band=0 重新跑一遍 score_band 过滤（纯 JS 计算，不调后端）
- 在同一表格中展示两列：实盘持仓 vs 无 band 虚拟持仓
- 差异以高亮标注——差异大说明 band 影响力强，差异小说明当前分数结构本身就足够分化
- 这是理解 BS 参数效果的最佳入口

## 改动范围

- `templates/tuner.html`：调仓快照增加反事实列 + JS 模拟 logic
