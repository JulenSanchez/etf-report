# REQ-177 多因子量化打分系统

> 立项：2026-04-25 | 最后更新：2026-04-27 | 状态：设计完成，待拆分实施

## 1. 背景与动机

用户从"简单回测"升级为正规的多因子每日打分系统。目标是对 25 支 ETF 选股池统一打分，输出 Top-6 持仓建议 + 仓位权重 + 周度条件单，替代纯主观决策。

## 2. 设计共识（v5 — 2026-04-27 多轮讨论定稿）

### 2.1 核心原则

- **一套逻辑，没有旁路**：所有行为（选股、仓位、现金比例、风控挂单）都来自同一个打分公式的正向计算或逆向推导
- **边做边优化**：先用简单方案跑一版看收益率，再迭代

### 2.2 选股池

25 支 ETF（用户富途自选全量），详见附录 A。

### 2.3 三因子（+估值）

全部输出连续浮点数，截面标准化到 0-100：

| # | 因子 | 计算 | 频率 | 角色 |
|---|------|------|------|------|
| F1 | 周线 EMA 偏离度 | (price - EMA_20w) / EMA_20w | 周线 | 趋势方向锚 |
| F2 | 日线 RSI(14) | 趋势自适应阈值（EMA 状态调 RSI 中值锚） | 日线 | 均值回归 |
| F3 | 方向性量比 | mean(上涨日成交额) / mean(下跌日成交额)，N 日滚动 | 日线 | 量价共振 |
| F4 | 估值百分位 | PB/PE 百分位（依赖 REQ-178 补齐 25 支） | 日线 | 基本面锚 |

### 2.4 偏好加成

有色(512400) / 创新药(513120) / 非银(512070) 综合分 +bias（回测标定，初估 +3~5 on 0-100 scale）。
效果：正常波动中保持席位稳定，技术面真崩时照样出局。

### 2.5 选股

- 常规 Top-6
- 特殊 Top-8（定义后续）

### 2.6 仓位

```
综合分 → 得分加权（算术 or softmax，回测对比）
       → × 信心函数 f(score)
       → 离散化到 5% 档位
```

**信心函数**（分段平方，方案 C）：
```
f(x) =
  0                           if x < 25 (dead_zone)
  ((x - 25) / 40)²           if 25 ≤ x < 65 (full_zone)
  1                           if x ≥ 65

参数 dead_zone=25, full_zone=65（初始值，回测后校准）
```

效果：牛市总仓位 80-95%，震荡市 40-60%，熊市 10-30%，极端 <25 分全现金。

### 2.7 风控挂单（周度）

每周末输出一次，同一套打分公式逆运算：
- **Top-6 持仓**：止盈价 + 止损价（反推"收盘价到多少时该 ETF 分数跌出 Top-6"的临界价）
- **第 7-8 名候补**：左侧买入价（限价单）+ 右侧买入价（突破买入单）
- 量比用近期均值假设（条件单无法挂成交额信息）
- 定位：添头级别，应对黑天鹅的基础保障

### 2.8 调仓节奏

- 常规：每周五收盘后算分 → 周一开盘执行
- 条件单：周末设好，一周不动

### 2.9 黑天鹅兼容

全市场暴跌 → 25 支 ETF 三因子同时恶化 → 信心函数压低绝对仓位 → 条件单批量触发 → 等效全卖。无需独立熔断模块。

## 附录 B：设计决策日志

| 日期 | 决策 | 理由 |
|------|------|------|
| 2026-04-27 | 从简单回测升级为多因子打分系统 | 用户希望接近正规量化做法 |
| 2026-04-27 | 三因子选定 EMA偏离+RSI+方向性量比 | 趋势+均值回归+量价三维度覆盖 |
| 2026-04-27 | RSI 趋势自适应阈值 | 上行趋势允许 RSI 高点，下行允许更低 |
| 2026-04-27 | 统一框架+偏好加成，不拆核心/卫星双层 | 用参数差异体现偏好，不用两套逻辑 |
| 2026-04-27 | 因子输出连续浮点而非离散档 | 3 因子太少，离散会丢信息 |
| 2026-04-27 | 仓位=得分加权×信心函数 | 信心函数让总仓位随市况自然涌现 |
| 2026-04-27 | 信心函数选分段平方（方案C） | dead_zone=25, full_zone=65，低分区谨慎、高分区饱和 |
| 2026-04-27 | 止盈止损=打分逆运算，非额外因子 | 系统只有一套逻辑的原则 |
| 2026-04-27 | 风控挂单简化为周度输出 | 用户手动调单，日更太麻烦 |
| 2026-04-27 | 现金比例由信心函数自然涌现 | 不设独立大盘判断模块 |
