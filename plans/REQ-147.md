## 动机

`index.html` 当前有两段 `<style>`：8-628 行（主样式 ~620 行）+ 707-1185 行（`debug-mode-style` ~480 行），合计 ~1180 行内联 CSS。抽离到独立文件：

- 首屏 HTML 体积显著下降
- CSS 可被浏览器独立缓存
- 调试态样式（`debug-mode-style`）单独成文件，易于维护

## 未覆盖 / 待决

- **[2026-04-20 评估新增] `#debug-mode-style` 动态引用探查（前置子任务）**：原 `<style id="debug-mode-style">` 带 id，需要先 grep 确认：
  - 是否有 `getElementById('debug-mode-style')` / `querySelector('#debug-mode-style')` 引用
  - 是否被动态注入 textContent / insertRule / cssText 等
  - 决策分支：
    - 若只是 id 占位、无动态操作 → 可直接改成 `<link rel="stylesheet" id="debug-mode-style" href="...">`
    - 若有动态 textContent 注入 → 必须保留空 `<style id="debug-mode-style">` 占位，或把相关逻辑改为用独立 style 节点 insertRule
- **[2026-04-20 评估新增]** 相对路径检查：确认原 CSS 中是否有 `url(...)` 引用（背景图 / 字体），抽到 `assets/css/` 后需调整相对路径
