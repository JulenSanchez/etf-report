## 背景

REQ-367（CAPM 残差 F7）被证伪并清退。但 9.24 行情暴露出 F7 的深层问题：

F7 无法区分"个股极端事件"和"全市场共振"。全市场齐涨齐跌时，所有 ETF 的 F7 Z-score 同时爆炸，map_f7 线性外延对每一支扣重分，F7 反客为主——选股退化为"在一堆负分里找不怎么负的"。

research 文档确认 F7 在个股异动检测中表现优秀（2024.7 纳指抄底、2025.1 卫星逃顶、2025.2 有色逃顶、2025.4 通信逃顶），9.24 是 F7 在"全市场共振"场景下的系统性误判。

## 设计

共识门控公式：

```
sigma_pool = std(all ETFs' raw F7 Z-scores)
damp = 1 / (1 + alpha * sigma_pool)
F7_contribution = map_f7(Z) * w7 * damp
```

- sigma_pool 大（个股异动）-> damp ~ 1 -> F7 正常工作
- sigma_pool 小（全市场共振）-> damp 小 -> F7 静音 -> F1/F3 接管
- alpha=0 等价于当前 gam-0（向后兼容）

不动 F7 公式本身，不动 map_f7。只在 composite 合成时加一个乘数。

alpha sweep: 0, 0.5, 1, 2, 5, 10, 20
通过标准: 6Y AR >= 107.4%（不退化），9.24 期间 F7 让位

## 实验结果 (2026-07-13)

**结论：共识门控被实验证伪。**

alpha sweep 在固定基线窗口 (2020-06-01 ~ 2026-06-01) 上运行：

| alpha | 6Y AR | 6Y MDD | 9.24 AR | 9.24 MDD |
|-------|-------|--------|---------|----------|
| 0 | 107.4% | -37.0% | 598.8% | -9.1% |
| 0.5 | 101.0% | -36.6% | 307.9% | -17.3% |
| 1 | 98.3% | -37.9% | 353.8% | -18.6% |
| 2 | 88.4% | -39.9% | 131.6% | -19.1% |
| 5 | 81.1% | -41.6% | - | - |
| 10 | 69.2% | -56.8% | - | - |
| 20 | 61.3% | -54.7% | - | - |

**任何 alpha > 0 都同时降低 AR 和恶化 MDD**——即使在 9.24 全市场共振期间。原假设"F7 在共振期反客为主"是错的：F7 在 9.24 的激进行为实际上是正确的（全市场动量真实且可获利）。
