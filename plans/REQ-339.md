# REQ-339: Stage B 融资利息 — 逐日计提

**优先级**: 🔴 P0 | **目标版本**: v3.11.0 | **状态**: ✅ 完成 (2026-07-02)

## 方案

参考交易佣金模式：每日检查 cash 是否为负（融资状态），若为负则计提利息并直接从 cash 扣除。不引入额外可变参数，不创建 margin_debt 独立变量。

```
if cash < 0: cash -= (-cash) * rate/365
```

## 已知局限

- 利息按交易日计提（252天/年），日利率用 rate/365（日历年基准），略低估实际年化
- 不追踪 margin_debt 独立科目，负 cash 即代表融资余额
- maintenance_ratio / margin call 留待 REQ-340
