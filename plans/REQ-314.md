# REQ-314: AccountState 序列与维持担保比例

**优先级**: 🟡 Normal | **目标版本**: v3.10.0 | **状态**: backlog | **创建**: 2026-06-24

## 目标

每日回测输出完整 AccountState 快照序列，实时追踪账户健康度。

## 规格

```
AccountState
  date, cash, long_market_value, gross_exposure, net_exposure
  net_liquidation_value, margin_debt, interest_accrued
  collateral_value, maintenance_ratio, available_buying_power
  margin_call_flag, forced_liquidation_flag
```

关键公式：
- NLV = cash + long_market_value - margin_debt
- collateral_value = long_market_value * haircut + max(cash, 0)
- maintenance_ratio = collateral_value / margin_debt (margin_debt > 0 时)
