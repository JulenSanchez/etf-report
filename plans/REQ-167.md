# REQ-167: 主题切换器

**优先级**: 🟢 Low
**状态**: wishlist
**创建日期**: 2026-04-21

## 背景

REQ-165/166（涨跌配色反转/蓝橙脱钩）废弃后的正式继承者。核心教训：配色是整体设计问题，不能局部改。

## 方案

技术路径：
1. 所有写死 hex 抽成 CSS variables（`--up-color` / `--down-color` / `--bg-base` / `--panel-surface` / `--text-primary` / `--rating-high` / `--info-neutral` 等）
2. 提供 3-5 套预设主题（富途夜间/Bloomberg/FT 日间/Linear/极简白）
3. `index.html` 加主题切换控件 + localStorage 持久化
4. ECharts 实例 theme 热切换（`echarts.use` 或 `init(option, theme)`）

## 触发时机

等当前开发重心收尾后重拾。未来需要一次真正的"视觉重设计"投入，建议单独立项彻底做。
