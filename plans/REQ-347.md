# REQ-347: Tuner 指标卡片精简与重排

**优先级**: 🟠 P1 | **目标版本**: v3.11.0 | **状态**: ✅ 完成 (2026-07-02)

## 背景

Tuner 原有 15 张指标卡（Page 1 × 10 + Page 2 × 5），存在信息冗余、分组不清、翻页交互增加认知负担等问题。本轮系统性重排。

## 改动

### 删除

| 卡片 | 原因 |
|------|------|
| 超额收益(沪深300) | gam-0 碾压基准，无信息量 |
| 均暴露 | 杠杆贡献卡片已覆盖，数值本身无决策意义 |
| >100%天数 | 同上 |
| 最差单日 | 同上 |

### 合并

| 前 | 后 | 显示格式 |
|----|-----|---------|
| 胜率 + 赔率（两张卡） | 胜率/赔率（一张卡） | `62% / 2.1x` |

### 新增

| 卡片 | 来源 |
|------|------|
| 杠杆贡献 | 从正式页借用（`leverage_contribution_pct`） |

### 重命名

| 前 | 后 | 原因 |
|----|-----|------|
| 融资压力 | 融资利息 | 与"交易佣金"对仗 |

### 翻页

两页合并为一页，删除 `flipMetricsPage` 逻辑和箭头。

### Tooltip 统一

- 全部使用公式右侧（无"X = "前缀）
- 删除所有主观判定（">2 为优"等）

## 最终布局

```
[年化收益] [总收益] [杠杆贡献] [融资利息] [交易佣金]     ← 收益与成本
[Sortino ] [Calmar] [最大回撤] [胜率/赔率] [ 换仓率 ]     ← 风险与行为
```

- Row 1：钱——收益、杠杆贡献、成本
- Row 2：质——风险调整指标、策略行为

## 不做的

- Sortino/Sharpe 翻转机制保留（用户要求恢复）
- 换仓率保留（用户主观认为重要）

## 改动范围

| 文件 | 改动 |
|------|------|
| `templates/tuner.html` | 卡片 HTML + JS + CSS 全部变更 |
