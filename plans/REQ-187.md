# REQ-187: 残差动量因子（Residual Momentum）

**优先级**: 🟡 Medium
**状态**: done
**创建日期**: 2026-05-06
**完成日期**: 2026-05-07

## 动机

传统动量因子（F1 EMA偏离度）混合了市场 beta 和个股 alpha，残差动量通过回归剥离市场成分，只保留"跑赢/跑输大盘的那部分动量"。学术研究表明残差动量比原始动量更稳定、换手更低。

## 方案

在 `quant_factors.py` 中实现 `factor_residual_momentum`：
1. 取最近 N 周（reg_window=12）的 (ETF周收益, HS300周收益) 对
2. OLS 回归: r_etf = alpha + beta * r_hs300
3. 残差 = r_etf - (alpha + beta * r_hs300)
4. 残差动量 = 最近 M 周（mom_window=12）残差累积和

映射函数 `map_f1_residual`：sigmoid，sensitivity=5.0（比 F1 的 8.0 更敏感，因残差量级更小）。

## 结论

**残差动量不纳入策略。** 全面负贡献，无论比例削减还是从w3抽取，w1r越高表现越差。

失败原因分析：
1. **截面太窄** — 25支ETF之间beta差异不大，回归残差空间太小，无法产生有效区分度（学术研究通常在数百支股票截面上验证）
2. **信号重叠** — F1(EMA偏离度)已经直接捕获了趋势信息，残差动量试图用不同角度捕获同类信息，反而引入噪音
3. **周收益负自相关** — 后视镜调研(REQ-189)已证明周收益lag-1自相关为负(-0.144)，这意味着"过去跑赢→未来跑赢"的动量延续假设在周频不成立

代码已保留在 `quant_factors.py` 中（`factor_residual_momentum` + `map_f1_residual`），tuner 也支持 `w1r` 参数，默认权重=0不影响现有策略。
