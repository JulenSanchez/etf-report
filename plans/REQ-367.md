## 背景

F7 是自我参照动量因子。全市场齐涨齐跌时区分度丧失，250d σ 被趋势污染。

最初考虑新建 F8，经推演发现更简单的方案：F7 公式中 r_ETF 替换为 r_ETF - β × r_bench。当 benchmark=None 时 β=0，完全等价当前 F7。无需新因子、新权重、新函数名。

## 设计

### 公式

```
if f7_benchmark_df is None:
    residual = r_ETF          # β=0, 当前 F7
else:
    β = rolling Cov(r_ETF, r_bench) / Var(r_bench), 250d
    residual = r_ETF - β × r_bench

cum_20d = Σ(residual, 20d)
F7 = (cum_20d - μ_250d) / max(σ_250d, sigma_floor)
```

**一个因子，两种模式。** benchmark=None → 当前 F7 行为不变。benchmark='000300' → CAPM 残差模式。
