## 动机

现有 CSS 中 `backdrop-filter: blur(10px)` + `rgba(255,255,255,0.05)` + `border-radius: 20px` + `border: 1px solid rgba(255,255,255,0.1)` 这套"玻璃卡片"样式在全局至少重复数十处。抽成 `.glass-card` utility class 后：

- CSS 体积下降
- 改动单点（例如调 blur 强度）一处生效
- 可变种（`.glass-card--sm` / `.glass-card--accent`）

## 实施步骤

### Step 1: 统计现有变种
- grep 所有 `backdrop-filter` / `rgba(255,255,255,0.0x)` 出现位置
- 归类：主卡片 / 次卡片 / 强调卡片

### Step 2: 设计 class 层级
- `.glass-card`（基底）
- 修饰符：`--sm` / `--accent` / `--subtle`

### Step 3: 替换 + 视觉回归
- 逐处替换，每替换一组截图对比
- 保证视觉 100% 一致
