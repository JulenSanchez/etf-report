## 动机

现有所有优化指标（Sharpe、Sortino、总收益、MDD）都是单期/单路径指标——它们衡量"这条历史路径上发生了什么"，不衡量"这个策略在统计学上有多大崩盘风险"。

凯利准则指出：真正的优化目标应该是**几何增长率（对数财富期望）最大化**，而非算术平均收益最大化。过度下注（f > 2f*）会导致期望增长率变负、无限重复后必然归零。

在 ETF 轮动策略中，「下注比例」= CS 参数。CS 越高，单 ETF 仓位越集中。需要回答：**在历史日收益序列的 bootstrap 重采样下，什么 (C, CS) 组合既不被单一历史路径绑架、又不会在坏的排列组合中崩盘？**

## 核心洞察

1. **单条历史路径的 ln(final) 排名 = 总收益排名**（ln 是单调函数），不是凯利分析
2. **Bootstrap 重采样 1000 条替代路径后的中位数 ln(final) 排名** ≠ 总收益排名——高波动策略分布更分散，中位数被拉低
3. 凯利最优 = bootstrap 中位数几何增长率最大 → 这才是"不会被坏运气杀死"的参数

## 方法

### Step 1: 数据准备
- 用已有 42 组 (C, CS) 网格（`cs_grid_search.json`），或重新生成 C × CS 回测
- 每组提取 6Y 日频 NAV 序列 → 日收益率序列

### Step 2: Bootstrap
- 对每组 (C, CS) 的日收益率序列，随机有放回重采样 1000 次
- 每条 bootstrap 路径从初始本金 1.0 开始，按重排后的日收益率复利到终点
- 记录 1000 个终值

### Step 3: 指标提取

| 指标 | 含义 |
|------|------|
| 中位数终值 | 替代历史中"最可能"的结果 |
| 5% 分位数终值 | 最差 5% 情况的底线 |
| 毁灭概率 | 终值 < 初始本金的比例 |
| 几何增长率中位数 | `median(ln(final))` / T |

### Step 4: 输出
- (C, CS) → 中位数终值/几何增长率 热力图
- (C, CS) → 毁灭概率 热力图
- 凯利最优 (C, CS)：中位数几何增长率最大
- 与 preset1（C=0.5, CS=10）对比

## 讨论记录

### 2026-05-27: 立项讨论

凯利公式：`f* = (bp - q) / b`，最大化长期几何增长率。关键性质：
- f < f*：增长偏慢但安全
- f = f*：几何增长率最大
- f* < f < 2f*：增长变慢，波动更大
- f > 2f*：期望增长率为负，必然归零

在策略中，CS 就是 f 的代理——控制仓位集中度。高的 CS 是"更大的赌注"。

用户观察：C=0.5, CS=0.6 时 Sortino=2.97 最优（1Y 窗口），远低于 preset1 的 CS=10。这可能是**短期窗口的凯利最优远低于长期窗口**——因为短期窗口的"硬币概率"不稳定性更高，需要更保守的下注。

### 未来方向（非穷举）

- Bootstrap 是探索凯利的一种方式，但不是唯一
- 可能的其他方向：蒙特卡洛参数化、滚动窗口破产概率、最大回撤持续时间分析
- 与 REQ-249（极端持仓至暗时刻）存在概念关联：都是关注"坏情况"
