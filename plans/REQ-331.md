# REQ-331: α 节奏热力带 — 策略 vs 自身趋势线

**优先级**: 🟠 P1 | **目标版本**: v3.11.0 | **依赖**: 无 | **状态**: 需求已确认，实现中

## 背景

Tuner 页 nav-chart 已有"仓位集中度"热力带，用绿→黄→橙→红四段色阶标记每个持仓周期的最大单票仓位。但缺少对**超额收益时间分布**的直观展示——策略哪些阶段 alpha 积累快于自身平均，哪些阶段慢于平均。

## 目标

新增"α 节奏"热力带：将策略净值与自身的恒定 CAGR 趋势线比较，偏离趋势的程度用色阶标记。绿色 = 超前趋势（alpha 积累快于平均），红色 = 落后趋势。

## 核心设计

### 公式：自身趋势偏离

```
T = 总交易日数
CAGR_daily = (S(T) / 100)^(1/T) - 1
Trend(t) = 100 × (S(T) / 100)^(t/T)          ← 从起点到终点的指数曲线

Deviation(t) = (S(t) / Trend(t) - 1) × 100   ← 相对偏离，百分比单位

rollingExcess[i] = avg(Deviation[i-W+1 .. i])  // W = dates.length / 12, min=10
```

- 起点和终点严格为 0（S(0)=Trend(0)=100, S(T)=Trend(T)）
- 中间围绕 0 分布，天然有正有负
- 不依赖 HS300，纯自比较

### 色阶

| 区间 | 颜色 | 含义 |
|------|------|------|
| < -20% | `rgba(239,68,68,0.12)` 红 | 显著落后趋势 |
| -20% ~ -7% | `rgba(249,115,22,0.10)` 橙 | 温和落后 |
| -7% ~ +7% | `rgba(250,204,21,0.08)` 黄 | 接近趋势 |
| > +7% | `rgba(34,197,94,0.08)` 绿 | 显著超前趋势 |

> 阈值为初始默认值。实现后根据 console 分布统计调整。

### 滚动窗口

```
W = Math.round(tradingDays / 12)，下限 10
```

| 回测周期 | W |
|----------|----|
| 1 年 | ~21 |
| 3 年 | ~63 |
| 6 年 | ~125 |

滚动平均门槛：`cnt > 0`（无最小填充要求，第一天即着色，与集中度热力带行为一致）。

## UI 设计：ECharts 图例互斥切换

### 方案

- **删除** `#kline-freq-tabs` 中的集中度/α HTML 按钮和分隔符，恢复为只有日K/周K
- 两个热力带 series **同时存在于 ECharts 中**，各自有 `name: '仓位集中度'` 和 `name: 'α 节奏'`
- ECharts legend 自动显示两个条目
- 监听 `legendselectchanged` 事件，实现互斥：

| 用户操作 | 结果 | 状态 |
|---------|------|------|
| 点击"仓位集中度"（当前隐藏→显示） | 显示集中度，隐藏 α | 01 |
| 点击"α 节奏"（当前隐藏→显示） | 显示 α，隐藏集中度 | 10 |
| 点击当前活跃的条目（显示→隐藏） | 两个都隐藏 | 00 |
| 两个都隐藏时点击任一条目 | 只显示被点击的那个 | 01 或 10 |

**11（两个同时显示）被 `legendselectchanged` 事件拦截，永远不允许。**

### 优势

- 零额外 HTML 元素——不需要按钮、切换器、分隔符
- 图例即控件——用户已经在看图例，操作就在视觉焦点上
- 互斥逻辑干净——3 行 JS 的 legendselectchanged handler
- 图例条目本身就是热力带的颜色说明

### 默认状态

初始加载时两个热力带都构建并加入 series，但 α 节奏的 markArea 数据为空（`heatBandMode === 'concentration'` 时 `alphaSignals` 为 null）。用户点击图例中的"α 节奏"后，触发重算 + 显示。

> 简化方案：两个热力带首次渲染时都有数据，互斥由图例控制。不再需要 `heatBandMode` 全局变量。

## 验收

1. 默认显示"仓位集中度"热力带，ECharts 图例中有"仓位集中度"和"α 节奏"两个条目

5. 两个热力带从不同时显示（11 状态不存在）
6. `#kline-freq-tabs` 中只有日K/周K按钮，无集中度/α 按钮
7. 周期切换（1Y/3Y/6Y）→ 正常
