# 量化估值因子入门

> 为什么机构都在讲"因子"？散户也用得上吗？这份文档把学术脉络、主流构造方法、常见陷阱一起讲清楚，不展开数学细节。

---

## 一、学术脉络：估值因子是怎么被"发现"的

### 1949 — Benjamin Graham《智慧投资者》

- "市场是投票机也是称重机"
- 提出**安全边际**概念（PE 10 倍以下买入）
- 这是价值投资的哲学起点，但还不是"因子"

### 1964 — Sharpe CAPM 模型

- 资本资产定价模型
- 股票收益只取决于**市场风险 β**（对大盘的敏感度）
- "所有其它因素都是噪声"

### 1992 — Fama-French 三因子（分水岭）

**核心发现**：除了市场风险 β，还有两个因子能**显著**解释股票长期收益：

| 因子 | 英文 | 含义 |
|-----|------|------|
| Size（市值） | SMB（Small Minus Big） | 小盘长期跑赢大盘 |
| Value（价值） | HML（High Minus Low B/M） | 高账面市值比（低 PB）跑赢低 B/M（高 PB） |

这就是**估值因子**的学术诞生：

> 便宜的股票长期跑赢贵的股票

**意义**：价值投资从"哲学"变成了"可量化、可验证的统计事实"。

### 1997 — Carhart 四因子

在 Fama-French 三因子基础上加入**动量（Momentum）**因子。过去 12 个月涨得多的股票短期倾向于继续涨。

### 2015 — Fama-French 五因子

继续扩展，加入：

| 因子 | 含义 |
|-----|------|
| RMW（盈利质量） | Robust Minus Weak，高 ROE 跑赢低 ROE |
| CMA（投资风格） | Conservative Minus Aggressive，低投资扩张的跑赢高的 |

**核心洞察**：
- 估值因子（HML）仍然保留
- 但**单看估值不够**，需要**盈利质量**配合
- 这解释了"成长股贵但合理"的现象（ROE 高可以支撑高 PB）

## 二、业界 5 种估值因子构造方法

### 方法 A：截面百分位（Cross-Sectional Rank）

**思想**：在**同一时点**横向排序

```
某指数 PE = 30
全 A 股 PE 中位数 = 22
→ 该指数在全市场截面 PE 百分位 = 72%
```

- **优点**：无需历史序列
- **缺点**：不回答"和自己过去比"
- **适用**：**选股**（挑便宜的），不适合判断"当前水位"

### 方法 B：时间序列百分位（Time-Series Rank）

**思想**：取过去 N 年每日 PE 序列排序，当前值插进去看落在几 %

- **优点**：直接回答"现在贵不贵"
- **缺点**：需要 **10 年历史序列**（工程上最大挑战，因为国内付费 API 贵，免费接口历史短）
- **适用**：集思录、理杏仁、韭圈儿的主流做法

### 方法 C：Z-Score 标准化

```
Z = (当前 PE - 均值 μ) / 标准差 σ
```

- **优点**：不同量纲的指标可以合成（比如 PE 的 Z 分 + PB 的 Z 分 = 综合估值分）
- **缺点**：PE 长尾分布，Z-Score 在非正态下失真
- **适用**：多因子合成的中间量

### 方法 D：锚点法（Anchor-Based）

**思想**：为每个指数/行业设立历史代表性锚点，当前值插值到百分位

例：
```
有色金属 PB 锚点：
  极低 1.2（2020 年疫情低点）
  低 1.8（2016 年底）
  中位 2.5
  高 3.5（2018 年上半年）
  极高 5.0（2021 年锂铜大牛市顶）

当前 PB = 2.0
→ 在 "低→中位" 之间插值
→ 约 35 百分位
```

- **优点**：**无历史序列也能算**（新指数友好）
- **缺点**：锚点主观，需人工维护
- **适用**：指数发布时间短、结构变迁大（需要专家校准）的场景
- **使用方**：FoF 投顾、券商行业配置月报、雪球大 V 估值日历

### 方法 E：盈利质量调整（PEG / ROE 加持）

**PEG**（Peter Lynch 1989）：
```
PEG = PE / 盈利增速
< 1 被低估, = 1 合理, > 1 高估
```

**ROE 调整**：
```
公允 PB = f(ROE, 资本成本)
ROE=20%, 资本成本=10% → 公允 PB ≈ 2
```

- **优点**：**惩罚价值陷阱**（周期股低 PE 但盈利增速差 → PEG 仍然不好看）
- **缺点**：需要盈利增速 / ROE 数据（比 PE 更难拿）
- **适用**：成长股、周期股估值

## 三、机构实操口径

### 中信证券行业配置模型（公开研报）

- 用 **PB 百分位（10 年）** 给行业打分 1-5
- PB < 20% 百分位：加配
- PB > 80% 百分位：减配
- 等价于**方法 B**

### 朱雀基金、兴证全球

- 不同行业用不同主指标（宽基看 PE、金融周期看 PB、创新药看 PS 等）
- 金融类特别强调 **PB-ROE 组合**

### 个人项目的现实路径

学术纯粹的方法 B 需要 10 年历史序列，免费数据源往往**拉不到**。现实做法：

1. **初期**：用方法 D（锚点）给一个合理但不精准的百分位
2. **每日增量累积**：把免费接口每天返回的数据追加到本地 CSV
3. **到一定样本后过渡**：100 天 → 500 天 → 完全按方法 B 运行

这就是**混合方法**（B')：锚点兜底，历史自然积累后自动切换。

混合切换规则示例：

```
n < 100 天 → 纯锚点
100 ≤ n < 500 天 → 锚点 + 历史加权混合（权重随 n 线性过渡）
n ≥ 500 天 → 纯时间序列百分位
```

## 四、PB-ROE 框架：超越单一估值

### 为什么单看 PB 不够

- 银行 PB 常年 0.5-0.8，看起来很便宜
- 日本银行 PB 失落 30 年一直 <1，但不是机会
- 关键差异：**能不能把净资产有效变现收益**

这就是 **ROE 维度**。

### PB-ROE 散点图

```
Y 轴: PB
X 轴: ROE

ROE 越高的公司 "应得" 更高的 PB。连一条趋势线:
- 趋势线上方: 估值偏高 (ROE 撑不起 PB)
- 趋势线下方: 估值偏低 (ROE 相对 PB 是便宜的)
```

这是 **兴证全球、富国**等价值派基金经理公开提过的框架。

### 实战示例

```
公司 A: PB=3, ROE=20% → PB/ROE=15, 合理
公司 B: PB=5, ROE=10% → PB/ROE=50, 贵
公司 C: PB=0.8, ROE=3% → PB/ROE=27, 便宜不了 (价值陷阱)
```

## 五、因子投资的几个反直觉现象

### 现象 1：价值因子长期跑赢但短期跑输

- 2000 年互联网泡沫：Fama-French HML 因子 **大幅跑输**
- 2020 年疫情后：**又一次大幅跑输**（成长股狂欢）
- 但拉长到 30 年，价值因子**年化超额收益 3-5%**

**启示**：因子不是短期择时工具，是长期配置依据。

### 现象 2：小盘因子在 A 股特别猛

美股 SMB（小盘）因子已经衰减，但 A 股因为散户占比高、流动性溢价明显，**小盘长期跑赢大盘**（2024 以前）。微盘股指数 2017-2023 年化 20%+，远超沪深 300。

### 现象 3：估值因子在熊市失效

- 熊市开始阶段：**便宜的跌得更多**（流动性差、机构抛售）
- 熊市末期：**便宜的反弹更猛**
- 这就是"先坠落再反弹"的钟摆

## 六、给个人投资者的启示

### 能做的

1. **用估值百分位当长期配置信号**（10% 以下加、90% 以上减）
2. **不同行业用不同指标**（周期 PB / 成长 PE / 红利 股息率）
3. **PB 配合 ROE 看**（避免价值陷阱）
4. **长期定投对标估值百分位低的指数**

### 不要做的

1. ❌ 用单日估值百分位做短期买卖
2. ❌ 无视行业特性一律套 PE 百分位
3. ❌ 认为"极度低估"就一定会反弹（可以更低）
4. ❌ 用自己拉的 1 年数据假装"历史百分位"（样本不够）
