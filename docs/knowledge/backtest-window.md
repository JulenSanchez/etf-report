# 回测窗口约定

> **触发词**: 用户说"起始日期""回测窗口""COVID""窗口选择""6Y"时参考本文。

## 主窗口：滚动 6 年

回测起始日期 = `today - 6 years`，即从今天往前推 6 年整。

```
当前: 2020-06-24 ~ 2026-06-24
更新: 每日报告自动向前滚动
```

**选择依据**：

1. **ETF 池子覆盖** — 2020 年中之前，A 股行业 ETF 数量不足当前的一半。用更早的数据等于假设策略可以轮动到当时不存在的产品，产生乐观偏差。
2. **市场风格连续** — 2020 年中以后的市场（注册制、量化崛起、北向资金、赛道轮动）与当前结构更接近。更早的市场是另一种生态。
3. **回测质量 > 覆盖长度** — 6 年包含 2021 科技回撤、2022 封城暴跌、2024-08 全球震荡，尾部事件已经足够丰富。

## 新冠窗口（2020-01 ~ 2020-06）

从 2020-01-01 起跑回测会额外包含新冠暴跌（2020-02 ~ 2020-03），使 gam-2 的 MDD 从 -21.69% 扩大到 **-31.80%**。

### 为什么不做硬约束

| 理由 | 说明 |
|------|------|
| **Singular event** | 新冠的驱动机制（全球停工、无限 QE、V 型反转）与 2021-2026 任何一次回撤都不同。它是另一种物理定律，不是"更严重的正常回撤"。 |
| **策略防御已生效** | -31.80% 是在 MA 趋势信号已经砍半仓（mbear=0.54）之后的数字。没有 MA 信号时满仓扛的 MDD 估计在 -40% 以上。策略的防御机制在工作，只是 MA19 的滞后在速跌中被放大。 |
| **ETF 宇宙虚构** | 2020 年 1 月实际存在的行业 ETF 数量远少于当前 54 支。回测中策略假设能轮动到当时不存在的创新药/芯片/通信 ETF——夸大了避险能力，新冠 MDD 本身包含乐观偏差。 |
| **复利代价** | 为满足 -25% 约束而调低杠杆，损失的 6Y 复利远超那 10pp MDD 的增量。 |

### 如何使用

```
新冠 MDD（~-32%）作为"极端压力测试参考值"，记录在案但不进入参数约束。
约束条件 MDD ≤ -25% 作用于 6Y 主窗口。

文档中应注明：
"策略内置 MA 趋势防御。在 MA 生效的前提下，新冠级别黑天鹅实测 MDD 约 -32%。"
```

### 仓位管理的真正防线

```
策略杠杆仓位 / 总资产 = 你的仓位占比
```

新冠级别的尾部风险由资产配置决策（你放多少钱进策略）来控制，不是由参数回测来控制。这个决策不在量化框架内。

## 历史回撤覆盖

6Y 窗口（2020-06 ~ 2026-06）包含的主要回撤事件：

| 时期 | 事件 | gam-2 表现 |
|------|------|-----------|
| 2020-08~09 | 复苏震荡 | MDD 峰值期，-21.69% |
| 2021-03~05 | 科技股回调 | NAV 187→173 |
| 2022-03~04 | 中国封城 | NAV 196→180 |
| 2024-01 | A 股季度暴跌 | NAV 303→265 |
| 2024-08 | 全球 carry trade unwind | NAV 358→282 |
| 2025-12 | 年末调整 | NAV 1056→932 |

所有事件中最大 MDD = -21.69%，发生在 2020 年秋。
