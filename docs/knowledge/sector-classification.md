# A 股行业分类体系对比

> 做行业比较、指数选型、配置分析时，常听到"申万一级""中证一级""万得一级""GICS"——它们什么关系？哪个最权威？

---

## 二、关键体系逐个解读

### GICS（Global Industry Classification Standard）

- **发布**: S&P（标普）+ MSCI（明晟）1999 年联合推出
- **结构**: 11 个一级（Sector）→ 24 个二级（Industry Group）→ 69 个三级（Industry）→ 158 个四级（Sub-Industry）
- **地位**: **全球资本市场的通用语言**
- **11 大一级**:
  1. Energy（能源）
  2. Materials（材料）
  3. Industrials（工业）
  4. Consumer Discretionary（可选消费）
  5. Consumer Staples（主要消费）
  6. Health Care（医疗保健）
  7. Financials（金融）
  8. Information Technology（信息技术）
  9. Communication Services（通信服务）
  10. Utilities（公用事业）
  11. Real Estate（房地产）

### Wind 一级行业

- **本质**：**GICS 的本地化实现**，11 大一级几乎照搬
- **强项**：Wind 终端内置，所有数据按此分类聚合，**机构端事实标准**
- **获取**：Wind 终端（¥30k+/年）独家；非 Wind 用户只能通过其他体系间接对齐

### 申万行业（SWS）—— **A 股散户事实标准**

- **发布**：申银万国 1999 年推出，沿用至今
- **31 个一级**：
  - 传统的："农林牧渔、采掘、化工、钢铁、有色金属、电子、家用电器、食品饮料、纺织服装、轻工制造、医药生物、公用事业、交通运输、房地产、商业贸易、休闲服务、综合、建筑材料、建筑装饰、电气设备、国防军工、计算机、传媒、通信、银行、非银金融、汽车、机械设备"
- **最新**：2021 年一次大调整，分成**纺织服饰、美容护理、社会服务、环保、石油石化**等新一级
- **为什么 A 股爱用它**：
  - 历史最久（28 年），散户语境深入
  - 覆盖**全 A 股每支股票**，归属确定
  - **中证指数有对应的 31 个申万一级行业指数**（000800-000863 系列），可以直接看到行业 PE/PB 历史
- **典型用法**：
  - "有色金属行业 PE"查 `中证申万有色金属指数 000819`
  - "银行业 PB" 查 `中证申万银行指数 000835`

### 中证一级

- **发布**：中证指数有限公司（上交所子公司）2008 年推出
- **11 个一级**：**直接对齐 GICS**
- **体系**：中证系列指数（000XXX / 930XXX / 931XXX 等）按此分类
- **特点**：覆盖 **A + H 股**（跨市场视角）

## 四、个人项目的实用建议

**结论**：用**申万一级** + **中证一级**组合，**等价于覆盖万得**。

### 工作流

```
① 先定行业（申万一级概念）
   例：有色金属 / 银行 / 非银金融 / 电子 / 医药生物

② 找对应指数代码
   申万一级指数（000819 有色 / 000835 银行 / 000837 非银）
   或 中证一级指数（中证 800 金融 000908 等）

③ 拉估值数据
   AKShare stock_zh_index_value_csindex(000819) → PE + 股息率
   Tushare pro.index_dailybasic("000819.SH") → PE + PB + 股息率

④ 分析 / 展示
   用申万口径给用户解读（散户更熟悉）
```

## 五、为什么行业分类会差异

不同体系选的维度不一样：

- **业务活动维度**（GICS / Wind / 中证）：按主营业务划分，"百济神州 → 医疗保健"
- **上市公司属性维度**（证监会）：按注册资料、监管分类
- **行业代表性维度**（申万）：偏重 A 股实际分布，如把"煤炭""有色"这种 A 股权重大的单独列一级

**实务中哪个用得多？**
- 卖方研报 / 投行报告 → **中信、申万**
- 监管统计 / 年报分类 → **证监会**
- 量化因子研究 / 机构投资 → **Wind（数据端）、申万（分析端）**
- 全球对比 → **GICS**

## 六、行业分类的三个经典陷阱

### 陷阱 1：同一家公司在不同体系里归属可能不同

例：阿里巴巴在 GICS 里是"Communication Services（通信服务）"，但在中证里是"可选消费"。研究时要明确用哪个口径。

### 陷阱 2：体系有调整历史

- 2018 年 GICS 把"电信"拆分并入通信服务，新增 Facebook / Alphabet 等
- 2021 年申万一级大修订，新增"美容护理""环保"等

做长期历史回测时注意"同一公司历史归属可能变"。

### 陷阱 3："行业"和"板块"不是一回事

- **行业**：基于主营业务分类（如申万一级）
- **板块**：基于主题 / 概念 / 热点（如"新能源车板块""AI板块"），多个行业成分混杂

散户口中的"电池板块"≠ 申万某个一级，通常是跨申万三级的概念指数。
