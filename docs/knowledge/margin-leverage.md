# 融资杠杆可执行上限

## 两融账户操作流程

A 股两融账户中，担保品买入和融资买入是分开的两步操作：

1. **担保品买入**：花光现金买入 ETF（持仓 100% NAV）
2. **融资买入**：系统根据担保品折算后价值，一次性算出最大可融额度

不存在无限循环迭代——融资买入只算一次。现金一旦换成 ETF，折算率从 100% 降到 r，可用保证金反而减少，因此不能再做任何担保品买入。

## 杠杆天花板公式

```
最大杠杆 = 1 + r    （r = ETF 折算率）
```

| r（折算率） | 最大杠杆 |
|-----------|---------|
| 100%（现金本身，不可维持） | 200% |
| 90% | 190% |
| 70% | 170% |
| 65% | 165% |
| 50% | 150% |

交易所硬上限 200%，维持担保比例 ≥ 130% 要到 433% 才触发，不构成实际约束。

## ETF 折算率实测

- **东方财富证券**（18.cn 公告）：几乎所有 ETF（股票/债券/跨境/商品）折算率统一为 **0.9**
- **华泰证券**：用户体感一致，ETF 折算率大部分 90%
- 个别高风险标的可能低于 90%，但池内主流 ETF 均可达到
