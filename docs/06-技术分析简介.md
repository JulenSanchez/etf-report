# 技术分析简介

> 散户最早接触股票时几乎都从 K 线、均线、MACD 开始。这份文档**不是技术派教程**，而是把技术分析的**语言、适用性、陷阱**讲清楚，让你知道什么时候可以看一眼、什么时候别迷信。

---

## 一、技术分析是什么

**定义**：**只用价格和成交量**预测未来走势的分析方法，不看公司基本面、不看估值。

**两大假设**：
1. **市场行为反映一切**（利好利空都已经在价格里）
2. **历史会重演**（类似形态往往导致类似后续走势）

**技术 vs 基本面 vs 量化**：

| 流派 | 关注 | 适用周期 | 典型工具 |
|-----|------|---------|---------|
| 技术分析 | 价量 | 短中期（分钟 - 数月） | K 线、均线、指标 |
| 基本面分析 | 公司盈利 / 估值 | 中长期（数月 - 数年） | PE / PB / ROE / DCF |
| 量化 | 因子 / 统计规律 | 跨周期 | 多因子模型、机器学习 |

这三者不互斥，**组合使用**是主流。

## 二、K 线：一切技术分析的基础

### 单根 K 线的含义

一根 K 线记录**一段时间**（日 / 周 / 月 / 分钟）的四个价格：

- **开盘价 Open**
- **最高价 High**
- **最低价 Low**
- **收盘价 Close**

### 阴阳怎么看

**A 股 / 港股习惯**：红涨绿跌（收盘 > 开盘 = 红 = 阳线）
**美股习惯**：绿涨红跌（相反）

国内散户日常所说的"红 K / 绿 K"就是 A 股口径。

### 典型单根 K 线形态

| 形态 | 形状 | 含义 |
|-----|------|-----|
| 大阳线 | 长实体红 | 强势买盘 |
| 大阴线 | 长实体绿 | 强势卖盘 |
| 十字星 | 开盘收盘接近 | 多空分歧，转折信号 |
| 锤头线 | 下影长、实体小 | 底部反弹信号 |
| 吊颈线 | 上影长、实体小 | 顶部回落信号 |
| 光头光脚阳线 | 开=最低、收=最高 | 极度强势 |

**重要**：**单根 K 线含义有限**，必须结合位置（高位 / 低位）和成交量看。

### 多根 K 线组合（经典形态）

- **早晨之星**（底部反转）：大阴 + 十字星 + 大阳
- **黄昏之星**（顶部反转）：大阳 + 十字星 + 大阴
- **红三兵**：连续 3 根阳线（多头强势）
- **黑三鸦**：连续 3 根阴线（空头强势）

这些形态在 A 股散户语境里用得很多，但**实战有效性远不如直觉那么高**（后面陷阱章节会讲）。

## 三、趋势与均线

### 均线（MA, Moving Average）

**公式**：过去 N 天收盘价的平均

```
MA20 = (C_t + C_{t-1} + ... + C_{t-19}) / 20
```

常用周期：
- **MA5 / MA10**：短期（周线级别）
- **MA20**：月线（最常用）
- **MA60**：季线
- **MA120**：半年线
- **MA250**：年线（牛熊分界）

### 均线的用法

**趋势判断**：
- 股价在 MA20 上 + MA20 向上 = 短期多头
- 股价在 MA20 下 + MA20 向下 = 短期空头
- MA5、MA10、MA20 **多头排列**（从上到下依次 MA5 > MA10 > MA20）= 强势

**黄金交叉 / 死亡交叉**：
- **金叉**：短期均线上穿长期均线（MA5 上穿 MA20 = 短期金叉）
- **死叉**：短期均线下穿长期均线

### 年线 MA250 的特殊地位

- A 股牛熊分界
- 股价站稳年线之上 = 技术面认为进入牛市
- 跌破年线 = 技术面转熊

但要注意，**指数级**的 MA250 更可靠，**个股级**的 MA250 经常被短期噪声破坏。

## 四、常见技术指标

### MACD（指数平滑异同移动平均线）

**本质**：两条均线的差值 + 其平均

```
DIF = EMA12 - EMA26
DEA = DIF 的 9 日 EMA
MACD 柱 = 2 × (DIF - DEA)
```

**用法**：
- DIF 上穿 DEA = 金叉买入
- DIF 下穿 DEA = 死叉卖出
- 柱状体翻红 / 翻绿 = 趋势转折信号
- 顶背离 / 底背离 = 高级信号

**优点**：趋势跟踪能力强
**缺点**：滞后明显，震荡行情假信号多

### KDJ（随机指标）

- 反映短期超买超卖
- K > 80 超买，K < 20 超卖
- 波动快，短线用多

### RSI（相对强弱指标）

- 公式衡量一段时间内上涨力度占比
- RSI > 70 超买，RSI < 30 超卖
- 比 KDJ 更平滑

### 布林带（Bollinger Bands）

- MA20 ± 2 倍标准差
- 股价触及上轨 = 短期压力
- 股价触及下轨 = 短期支撑
- 带宽收敛 = 即将选择方向

### 成交量（Volume）

**技术分析里最被忽视但最重要**的指标。

- **放量上涨**：真涨（有资金认可）
- **缩量上涨**：假涨（可能下跌）
- **放量下跌**：真跌
- **缩量下跌**：可能企稳

> 记住一句话：**"量在价先"**（成交量往往先于价格变化）

## 五、支撑位、压力位、趋势线

### 支撑位 / 压力位

- **支撑位**：股价多次跌到某价位就反弹 → 下方买盘密集
- **压力位**：股价多次涨到某价位就回落 → 上方卖盘密集
- 压力位被有效突破后往往**变成新的支撑位**（所谓"压力变支撑"）

识别方法：
- 历史 K 线的高点 / 低点聚集区
- 大成交量的成交密集区
- 整数价位（10 元、100 元这种心理价位）

### 趋势线

- **上升趋势线**：连接至少两个低点画线，线之下是跌破趋势
- **下降趋势线**：连接至少两个高点画线，线之上是突破趋势

### 经典形态

- **头肩顶 / 头肩底**：三个高点 / 低点，中间最高 / 最低 → 反转信号
- **双顶 / 双底（M 顶 / W 底）**：两次同样价位冲顶 / 探底 → 反转信号
- **三角形**：高点下移 + 低点上移 → 波动收敛后大方向突破
- **旗形 / 楔形**：趋势中继（趋势暂停后继续）

## 六、技术分析的几个经典陷阱

### 陷阱 1：大量事后归纳 + 生存者偏差

技术分析书里的图谱大多是**成功案例**。失败的形态（金叉后继续跌、头肩底不反转）往往不被展示。真实世界里，**任何形态都有 40-50% 的失败率**。

### 陷阱 2：小盘股和大盘股的信号完全不同

- 大盘股流动性好 → 指标平滑、信号可信
- 小盘股波动大 → KDJ / RSI 假信号满天飞
- 指数级的技术分析>>个股级的

### 陷阱 3：指标组合 ≠ 更准

"MACD 金叉 + KDJ 金叉 + 放量突破" 听起来很严谨。但多个指标来自**相同的价格数据**，高度相关，**组合不会大幅提高准确率**，反而会延迟信号。

### 陷阱 4：背离信号常"失灵"

理论上"顶背离"（价格新高但指标不新高）应该卖出。但 2020-2021 美股 / A 股牛市，**背离出现 5-6 次后行情才见顶**，过早卖出的都踏空。

### 陷阱 5：A 股涨跌停机制扭曲形态

涨跌停限制下：
- 强势股 K 线极不规则（连板 + 一字板）
- 放量突破 / 跌破失真（没法正常成交）

### 陷阱 6：技术 + 基本面背离时

**技术面看好但基本面很烂**（如 ST 股操作、垃圾股脉冲）:
- 短期技术可能成立
- 但长期**一定回到基本面**
- 散户最容易在这种标的上爆仓

## 七、技术分析的合理用法

### ✅ 可以这样用

1. **择时**（什么时候买入 / 卖出某支你已经看好的股票）
2. **止损位**（设置在关键支撑位下方）
3. **辅助指数波动节奏**（看 MA20 判断短期趋势）
4. **观察成交量异常**（真的大资金进出）

### ❌ 不要这样用

1. ❌ **完全不看基本面只做技术**（长期必爆）
2. ❌ **把短期信号当作中长期依据**
3. ❌ **把 MACD 金叉当成买入保证**
4. ❌ **用小盘股 / 题材股的形态类比大盘股**

### 一个简化的个人框架

> **基本面定标的，估值定仓位，技术定时点**

- 先靠基本面排除烂公司 / 烂行业
- 靠估值决定加仓 / 减仓区间
- 靠技术面决定**具体哪一天**下单

## 八、ETF 适不适合用技术分析？

**ETF 技术面 vs 个股技术面**：

| 维度 | ETF | 个股 |
|-----|-----|-----|
| 流动性 | 多数够用 | 参差 |
| 涨跌停 | 有但很少触发 | 常见 |
| 突发事件 | 被成分股分摊（弱化） | 集中（放大） |
| 因子漂移 | 指数稳定 | 公司基本面易变 |
| 技术信号可靠性 | **相对更高** | 噪声大 |

**结论**：**ETF 是技术分析 + 基本面结合的良好标的**。指数成分股分散了个股风险，让趋势/形态更平滑。

尤其是宽基 ETF（沪深 300、创业板指）的均线信号，比单一个股可靠得多。
