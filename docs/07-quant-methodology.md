# 量化打分系统方法论

> REQ-177 M5.1 · 三因子 ETF 轮动策略 · 25 支选股池

## 1. 系统定位

一套**纯量化、纯规则驱动**的 ETF 轮动策略，目标：
- 周度频率调仓，适合兼职散户
- 不预测市场方向，只做截面排序（选强去弱）
- 通过信心函数自动控制总仓位（牛市满仓、熊市轻仓）
- 所有行为来自同一套打分公式，无人工判断

## 3. 三因子模型

### 3.1 F1: 周线 EMA 偏离度（趋势锚）

```
F1 = (close_weekly - EMA_20w) / EMA_20w × 100
```

- **含义**：价格偏离 20 周均线的程度（%）
- **正值**：高于均线 = 上行趋势
- **负值**：低于均线 = 下行趋势
- **权重**：35%（均衡模板）

### 3.2 F2: RSI 趋势自适应变换（均值回归）

```
RSI(14) → 根据趋势状态(F1正负)调整打分区间 → 0-100
```

- **上行趋势**：RSI 40-65 最佳（80-100分），>75 快速扣分
- **下行趋势**：RSI 30-50 最佳（70-100分），>60 大幅扣分
- **核心思想**：同一个 RSI=60，牛市里是正常，熊市里是过热
- **权重**：30%（均衡模板）

### 3.3 F3: 方向性量比（量价共振）

```
F3 = mean(上涨日成交额, 20日) / mean(下跌日成交额, 20日)
```

- **>1**：资金在上涨日更活跃 = 多头占优
- **<1**：资金在下跌日更活跃 = 空头占优
- **权重**：35%（均衡模板）

### 3.4 截面标准化

所有因子独立做 **百分位排名**（percentile_rank），映射到 0-100 分。

### 3.5 综合分合成

```
composite = w1×norm_F1 + w2×norm_F2 + w3×norm_F3 + bias_bonus
```

偏好加成（bias_bonus）：对 3 支偏好标的额外加分，使其在正常波动中保持席位稳定。

## 4. 选股与仓位

### 4.1 Top-N 选股

取综合分最高的 N 支（默认 6，保守型 4，激进型 8）。

### 4.2 信心函数（分段二次型）

```
f(score) =
  0                              if score < dead_zone (25)
  ((score - dead_zone) / range)² if dead_zone ≤ score < full_zone (65)
  1                              if score ≥ full_zone
```

- **dead_zone 以下**：信心为 0，不配仓
- **中间区**：二次上升（平方），低分段加仓保守
- **full_zone 以上**：满配

### 4.3 总仓位水平

```
total_target = min(0.95, avg_confidence × 1.2)
```

- 牛市平均信心高 → 总仓位 80-95%
- 熊市所有 ETF 分数低 → 总仓位可压到 10-30%
- **无需独立熔断模块**：全市场暴跌时信心自然压低

### 4.4 个股仓位

```
target_position[i] = (score[i] / sum(scores)) × total_target
→ 离散化到 5% 档位（round to nearest 0.05）
```

## 5. 调仓节奏

- **默认**：每周五收盘后计算 → 周一开盘执行
- **条件单**：基于风控挂单的止盈/止损价

## 6. 风控挂单（M4）

### 6.1 原理

F1 是唯一直接依赖价格的因子。假设 F2/F3 不变，反推：
- **止损价**：价格跌到什么水平时，该 ETF 综合分会跌出 Top-6
- **止盈价**：价格涨到什么水平时，综合分达到满配区（65分）
- **买入价**（候补）：价格涨到什么水平时，候补 ETF 能进入 Top-6

### 6.2 局限性

- RSI 和量比实际上也会随价格变动，临界价是**近似值**
- 仅供设置条件单参考，不是精确的交易价格
- 建议留 2-5% 安全边际
